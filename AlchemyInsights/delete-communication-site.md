# <a name="delete-a-communication-site"></a>Удаление сайта для общения

1. Перейдите на сайт.
    
2. Щелкните значок Параметры в правом верхнем углу.
    
3. Щелкните **Параметры сайта**. В разделе **сведения о сайте**выберите **удалить этот сайт**, а затем нажмите кнопку **Удалить**.
