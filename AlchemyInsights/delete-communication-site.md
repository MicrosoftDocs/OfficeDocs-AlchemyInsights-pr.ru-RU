# <a name="delete-a-communication-site"></a>Удаление связи сайтов

1. Перейдите на сайт.
    
2. Щелкните значок параметры в правом верхнем углу.
    
3. Выберите **Параметры сайта**. В разделе **Сведения о сайте**нажмите кнопку **Удалить этот сайт**и выберите команду **Удалить**.
