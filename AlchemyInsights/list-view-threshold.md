# <a name="list-view-threshold"></a>Пороговое значение представления списка

Пороговое значение этого списка позволяет сохранить производительность службы. Пороговое значение не может быть изменено в SharePoint Online. Чтобы избежать превышения порогового значения, см. раздел [Управление крупными списками и библиотеками в SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59).

Вы также можете получить эту ошибку при попытке поделиться или нарушить наследование папки. См. раздел [Ошибка предоставления наследования](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance).
