# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Устранение неполадок сценариев обновления — ошибки плитки

Подождите около 10-15 минут, чтобы обновление отображалось в плитках панели мониторинга. Если он по-прежнему не отображается, повторно закрепите визуализацию на панели мониторинга.
