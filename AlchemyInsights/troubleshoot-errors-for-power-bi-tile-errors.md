# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Устранение неполадок сценариев обновления — ошибки плитки

При возникновении ошибок обновления плитки в службе Power BI необходимо подождать около 10-15 минут, чтобы обновление отображалось в плитках панели мониторинга. Если он по-прежнему не отображается, повторно закрепите визуализацию на панели мониторинга.

Дополнительные сведения:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
