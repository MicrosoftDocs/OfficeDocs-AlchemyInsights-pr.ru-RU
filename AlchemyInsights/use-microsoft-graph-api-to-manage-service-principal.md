# <a name="use-microsoft-graph-api-to-manage-service-principal"></a>Использование API Microsoft Graph для управления основными службами

С помощью [API-API](https://docs.microsoft.com/graph/api/resources/serviceprincipal)для основных служб можно программным образом управлять экземплярами приложений и управлять тем, что приложение может делать в клиенте.

Дополнительные сведения о связи между приложениями и объектами-службами см. в приложениях и [объектах-службах в Azure Active Directory.](https://docs.microsoft.com/azure/active-directory/develop/app-objects-and-service-principals)
