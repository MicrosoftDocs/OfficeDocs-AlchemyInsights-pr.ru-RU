# <a name="edit-a-read-only-file"></a>Редактирование файла только для чтения

В библиотеках документов при нажатии кнопки на имя документа, программа Office может неожиданно открыть его как файл только для чтения. Чтобы открыть ее для редактирования, нажмите кнопку **Открыть** (трех точек) и нажмите кнопку **Изменить**
