# <a name="edit-a-read-only-file"></a>Изменение файла, доступного только для чтения

В библиотеках документов при щелчке имени документа приложение Office может неожиданно открыть его в качестве файла, доступного только для чтения. Чтобы открыть его для редактирования, нажмите кнопку **Открыть** (три точки), а затем нажмите кнопку **изменить.**
