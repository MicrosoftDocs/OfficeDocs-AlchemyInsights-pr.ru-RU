# <a name="creating-a-team-from-an-existing-team"></a>Создание команды из существующей команды

Некоторые клиенты сообщили о проблеме с созданием команды из существующей команды (или клонированием команды), когда они решили скопировать участников команды. Чтобы обойти эту проблему, не копируйте участников в процессе создания и не добавляйте участников после создания команды.

Дополнительные сведения см. [в теме "Создание команды из существующей команды".](https://support.microsoft.com/office/create-a-team-from-an-existing-team-f41a759b-3101-4af6-93bd-6aba0e5d7635)
