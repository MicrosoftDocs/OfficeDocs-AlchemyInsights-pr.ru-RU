# <a name="whats-a-hub-site"></a>Что такое центральный сайт?

Центральный сайт позволяет организации сайтов и предоставьте им распространенных навигации и фирменной символики. Для создания сайтов сервера-концентратора и укажите пользователей, которые можно связать их сайты центральный сайт будет Global или SharePoint "Администраторы". Дополнительные сведения в разделе [сервер-концентратор сайтов SharePoint новые материалы в Office 365](https://go.microsoft.com/fwlink/?linkid=869388).
