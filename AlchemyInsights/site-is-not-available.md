# <a name="site-is-not-available"></a>Сайт недоступен

К нескольким пользователям сайта SharePoint или OneDrive недоступна, возможны проблемы временной службы. Проверьте[Панель мониторинга работоспособности службы](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) для просмотра, если клиент является затронутые.
