# <a name="site-is-not-available"></a>Сайт недоступен

Если сайт SharePoint или OneDrive недоступен для нескольких пользователей, может возникнуть временная ошибка службы. Просмотрите [панель мониторинга работоспособности службы](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) , чтобы проверить, влияет ли клиент на работу.
