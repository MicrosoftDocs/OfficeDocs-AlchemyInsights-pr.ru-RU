1. <span data-ttu-id="eec0e-104">На [странице Домены](https://portal.office.com/adminportal/home#/Domains) выберите домен, а затем нажмите кнопку **Удалить домен**.</span><span class="sxs-lookup"><span data-stu-id="eec0e-104">On the [Domains page](https://portal.office.com/adminportal/home#/Domains), choose the domain, and then select  **Remove domain**.</span></span></br>

    <span data-ttu-id="eec0e-105">Подробные инструкции см. в статье [Как удалить домен](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain?view=o365-worldwide).</span><span class="sxs-lookup"><span data-stu-id="eec0e-105">For for more detailed instructions, see [How to remove a domain](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain?view=o365-worldwide).</span></span>
