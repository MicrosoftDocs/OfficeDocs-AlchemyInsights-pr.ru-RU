# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Использование общего почтового ящика на мобильном устройстве

Чтобы использовать общий почтовый ящик на мобильном устройстве, необходимо [скачать Outlook Mobile](https://products.office.com/outlook-mobile-for-android-and-ios) , а затем [Добавить общий почтовый ящик в Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
