# <a name="your-users-mailbox-is-being-created"></a>Идет создание почтового ящика пользователя

Для того чтобы новый почтовый ящик был готов к использованию, он может занять до часа.
  
Если почтовый ящик пользователя был подготовлен к работе в течение более часа, можно попробовать удалить и повторно создать пользователя, или воспользоваться параметром "помочь нам позвонить", чтобы поговорить с инженером службы поддержки.
