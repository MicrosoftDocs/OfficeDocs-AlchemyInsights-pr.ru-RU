# <a name="print-an-attachment"></a>Печать вложения

Чтобы распечатать вложений электронной почты, загрузка и сохранение вложений на локальном компьютере или OneDrive и откройте их с соответствующая программа.
  
1. Откройте сообщение с вложением, которые необходимо напечатать.
    
2. Ниже вложения выберите **загрузить**. 
    
3. После загрузки файла откройте его с нужной программы и выберите параметр для печати. Например откройте файл DOC в Word и распечатайте.
    
Для получения дополнительных сведений прочитайте [печати сообщения электронной почты и вложений в Outlook.com](https://go.microsoft.com/fwlink/?linkid=2021110&amp;clcid=0x409).
