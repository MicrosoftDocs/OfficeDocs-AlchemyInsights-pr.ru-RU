# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Отчет о совместном использовании файлов и папок на сайте SharePoint

Вы можете создать CSV-файл каждого уникального файла, пользователя, разрешения и ссылки на определенном сайте SharePoint или в библиотеке OneDrive. Это поможет вам понять, как используется общий доступ, и предоставляются ли файлы или папки гостям. Дополнительные сведения см. в статье [Отчет о совместном использовании файлов и папок на сайте SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).
