# <a name="perform-access-reviews-in-azure-active-directory"></a>Выполнение проверок доступа в Azure Active Directory

Назначенные рецензенты могут выполнять проверки доступа Azure Active Directory на членство в группах, доступ к приложениям и назначения ролей. Эти проверки помогают снизить риски безопасности, с которые может столкнуться организация, если у пользователей больше доступа, чем им нужно. Вы можете выполнять проверки доступа с помощью [My Apps](https://go.microsoft.com/fwlink/?linkid=2134605) или [My Access.](https://go.microsoft.com/fwlink/?linkid=2134505)
