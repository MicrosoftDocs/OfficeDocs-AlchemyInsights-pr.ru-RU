# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>С помощью общего почтового ящика на мобильном устройстве

Не может получить доступ к общему почтовому ящику из приложения Outlook, которая устанавливается на мобильном устройстве. Вы хотите, чтобы этот компонент? Пусть знать команды разработчиков Outlook. [Здесь голос](https://go.microsoft.com/fwlink/?linked=862116)!
  
Общего почтового ящика можно использовать только на мобильном устройстве, открыв браузера, вход в Office 365 и затем переход на Outlook в Интернете. Для получения инструкций в разделе [Add общего почтового ящика в Outlook в Интернете](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Если необходимо получить доступ к общим почтовым ящикам с помощью приложения Outlook на мобильном устройстве, можно создать группу Office 365 вместо использования общего почтового ящика. Для получения дополнительных сведений см [Сравнение групп](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx).
  
Дополнительные сведения: [Получение справки в приложение для Outlook для iOS и Android (en)](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
