# <a name="onedrive-not-responding"></a>OneDrive не отвечает

Для только что созданный учетных записей пользователи должны войти в портал Office 365 с помощью собственных учетной записи и щелкните элемент OneDrive, чтобы вручную запустить подготовки сайта OneDrive.
  
Тем не менее если уже выбираемого пользователем OneDrive и личных сайтов не подготовить к работе в течение 24 часов, войдите билетов поддержки за помощью.
