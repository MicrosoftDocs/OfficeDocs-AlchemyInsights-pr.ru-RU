# <a name="content-type-publishing"></a>Публикация типов контента

Если вы хотите, чтобы типы контента были доступны в Организации для всех пользователей, рекомендуется опубликовать их во всех семействах веб-сайтов. Чтобы опубликовать тип контента, ознакомьтесь со статьей [Публикация типа контента из концентратора публикации контента](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).
