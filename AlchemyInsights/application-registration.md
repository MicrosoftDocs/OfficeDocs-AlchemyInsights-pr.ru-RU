# <a name="application-registration"></a>Регистрация приложения

1. Регистрация приложения в [Azure Active Directory:](https://docs.microsoft.com/powerapps/developer/data-platform/walkthrough-register-app-azure-active-directory)в этом руководстве описывается регистрация приложения в Azure Active Directory, которая позволяет пользователю с учетной записью пользователя Power Apps подключаться к своей среде Microsoft Dataverse из внешних клиентских приложений с помощью проверки подлинности OAuth.
1. Зарегистрируйте приложение с помощью платформы удостоверений [Майкрософт.](https://docs.microsoft.com/azure/active-directory/develop/quickstart-register-app)В этом кратком примере вы зарегистрируете приложение на портале Azure, чтобы платформа удостоверений Майкрософт может предоставлять службы проверки подлинности и авторизации для вашего приложения и его пользователей.
