# <a name="add-a-content-type-to-a-list-or-library"></a>Добавление типа контента в список или библиотеку

1. Перейдите к  [параметрам для списка или библиотеки](https://support.microsoft.com/en-us/office/edit-list-settings-in-sharepoint-online-4d35793b-246e-42a3-990c-563a83795b7f).
2. В разделе  **типы контента**выберите элемент  **Добавить из существующих типов контента сайта**. Если  **типы контента**  не отображаются, выберите  **Дополнительные параметры**.
3. Выберите из списка  **выберите типы контента сайта**  из списка.
4. В списке Доступные типы контента сайта выберите имя нужного типа контента и переместите выбранный тип контента в список типы контента для добавления, нажмите кнопку Добавить.
5. Повторите шаги для дополнительных типов контента.

Дополнительные сведения см. в разделе  [Добавление типа контента в список или библиотеку](https://support.microsoft.com/en-us/office/add-a-content-type-to-a-list-or-library-917366ae-f7a2-47ad-87a5-9689a1884e60).
