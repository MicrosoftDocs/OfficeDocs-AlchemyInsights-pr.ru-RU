# <a name="stalleddueto-errors-during-migration"></a>Ошибки Сталледдуето во время миграции

Могут отображаться различные ошибки, начинающиеся с "Сталледдуето..." во время миграции в Exchange Online. Многие из этих ошибок вполне нормальны. Дополнительные сведения см в разделе [регулирование на основе ресурсов и определение приоритетов при миграции Exchange Online](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).
