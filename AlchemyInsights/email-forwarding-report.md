# <a name="email-forwarding-report"></a>Переадресация отчета электронной почты

Для просмотра отчета на сколько автоматической пересылки сообщений в организации, проверьте [полезные сведения о поток почты](https://support.office.com/article/beb6acaa-6016-4d54-ba7e-3d6d035e2b46) [безопасности Office 365 &amp; центре соответствия требованиям](https://protection.office.com/#/homepage).
  
Обратите внимание: Если это при первом выполнении входа в безопасности &amp; центре соответствия требованиям, может потребоваться обновить окно браузера через несколько минут, чтобы просмотреть все доступные параметры.
