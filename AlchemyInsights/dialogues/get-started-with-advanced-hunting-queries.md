# <a name="get-started-with-advanced-hunting-queries"></a>Начало работы с расширенными запросами на охоту

Исследователи безопасности Майкрософт регулярно делятся расширенными запросами на [охоту в GitHub.](https://go.microsoft.com/fwlink/?linkid=2144624)

Исследователи также предоставляют расширенные запросы на охоту для размещения действий и индикаторов, связанных с [возникающими угрозами.](https://go.microsoft.com/fwlink/?linkid=2145808) Эти запросы предоставляются в рамках отчетов по аналитике угроз в Центре безопасности Защитника Майкрософт.
