# <a name="avoid-multiple-signatures-from-being-added-in-an-email-conversation"></a>Предотвращение того, чтобы в беседе по электронной почте не добавлялись несколько подписей

Чтобы получить заявление об отказе или подпись, добавленные только один раз для каждого потока электронной почты, необходимо изменить параметр в Центре администрирования Exchange.

1. Перейдите в [Центр администрирования Exchange.](https://go.microsoft.com/fwlink/p/?linkid=2059104)
2. Choose **More options** to add an exception that looks for unique text in your disclaimer. Это гарантирует, что заявление об отказе будет добавлено только в исходное сообщение.
