<span data-ttu-id="86ab2-103">С помощью подчиненных меток вы можете группировать одну или несколько меток под родительской меткой, которая видна пользователю в приложении Office.</span><span class="sxs-lookup"><span data-stu-id="86ab2-103">With sublabels, you can group one or more labels below a parent label that a user sees in an Office app.</span></span> <span data-ttu-id="86ab2-104">Например, для метки **Конфиденциально** в вашей организации может использоваться несколько разных меток для определенных типов этой классификации.</span><span class="sxs-lookup"><span data-stu-id="86ab2-104">For example, under **Confidential**, your organization might use several different labels for specific types of that classification.</span></span>
