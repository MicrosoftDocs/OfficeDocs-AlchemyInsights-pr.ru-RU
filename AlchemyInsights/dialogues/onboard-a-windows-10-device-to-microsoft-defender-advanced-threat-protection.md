# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>На борту устройства Windows 10 в Microsoft Defender Advanced Threat Protection (MDATP)

MDATP поддерживает следующие средства развертывания и методы для Windows 10:

- Групповая политика
- Microsoft Endpoint Configuration Manager
- Управление мобильными устройствами, включая Microsoft Intune
- Локальные сценарии

Дополнительные дополнительные информации см. в таблице [Onboarding tools and methods for Windows 10 devices.](https://go.microsoft.com/fwlink/?linkid=2143460)
