# <a name="change-your-account-picture"></a>Изменить изображение учетной записи

1. Перейдите на [свой Microsoft сведения учетной записи](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Выберите **Изменить рисунок**. 
    
3. Выберите **новый рисунок**. 
    
4. Выберите изображение, которое нужно использовать и выберите команду **Открыть**. 
    
5. Перетащите или изменить размер поля изображения и выберите команду **Сохранить**. 
    
Подождите до 24 часов для изменения фотографий профиля или имя вступить в силу.
