# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Назначение псевдонима электронной почты основному адресу пользователя

Инструкции можно найти [в разделе Добавление другого псевдонима электронной почты для пользователя](https://docs.microsoft.com/office365/admin/email/add-another-email-alias-for-a-user).

**Примечание**. Если ваша организация находится в гибридном режиме, необходимо изменить основной адрес электронной почты в локальной организации Exchange. Дополнительные сведения см. [в статье Добавление и удаление адресов электронной почты для почтового ящика](https://technet.microsoft.com/library/bb123794.aspx).
