# <a name="credit-check-status-request"></a>Запрос состояния проверки кредита

Заказ — проверка ожидающие кредит. Это обычно выполняется в менее 24 часов, но может потребоваться до 48 часов. По завершении подписки состояние изменится из ожидающих активно и службы будут доступны для использования.
