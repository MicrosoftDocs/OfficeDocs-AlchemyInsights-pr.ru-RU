# <a name="profile-property-synchronization"></a>Синхронизация свойств профиля

Невозможно сопоставить дополнительные свойства импорта AD для синхронизации с Azure Active Directory с приложением профилей пользователей. Импорт AD ограничен предварительно настроенным набором свойств для обеспечения согласованной производительности задания таймера.
  
[Дополнительные сведения о свойствах профиля, которые синхронизируются](https://go.microsoft.com/fwlink/?linkid=875671)
