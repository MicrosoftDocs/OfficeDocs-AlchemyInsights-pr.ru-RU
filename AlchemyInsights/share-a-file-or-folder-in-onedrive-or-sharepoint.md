# <a name="share-a-file-or-folder-in-onedrive-or-sharepoint"></a>Совместное использование файла или папки в OneDrive или SharePoint

1. Откройте файл или папку, которую требуется общий доступ, выберите его и нажмите кнопку **общий доступ**.
    
2. (Необязательно) В диалоговом окне Отправить ссылку щелкните стрелку вниз, чтобы изменить тип ссылки. В группе **пользователей, которые желаете работы для этой ссылки**, доступа к данным, ссылку. Пользователи должны иметь возможность просмотра, но не для редактирования файлов, снимите флажок **Разрешить изменение** . 
    
    Нажмите **Применить**.
    
    > [!NOTE]
    > Если параметр общего доступа, который будет недоступен, необходимо убедитесь, что разрешено для вашей организации и семейства веб-сайтов. Для этого видеть [Включить внешний общий доступ к включено или отключено для SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866426). 
  
3. Введите имена людей, с которыми вы хотите поделиться с и нажмите кнопку **Отправить**.
