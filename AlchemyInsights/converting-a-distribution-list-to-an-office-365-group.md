# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Преобразование группы рассылки в группу Microsoft 365

Вы можете преобразовать группы рассылки в Группы Microsoft 365, если они соответствуют определенным критериям. Чтобы преобразовать проблемы преобразования группы или устранения неполадок, см. в списке рассылки обновления [в Microsoft 365 Groups in Outlook.](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)

У нас также есть сценарий, который можно запустить, чтобы помочь вам с проверкой права группы рассылки на обновление до [Microsoft 365 Group.](https://aka.ms/DLToM365Group)
