# <a name="block-email-forwarding"></a>Блокировка переадресации электронной почты

Чтобы отключить переадресацию электронной почты для определенного почтового ящика, ознакомьтесь со статьей [Настройка переадресации электронной почты](https://support.office.com/client/15abf81d-5c5d-49da-ac81-1b4daa1809f6).

Чтобы заблокировать переадресацию электронной почты для всей Организации, [множество способов блокировать автоматическую переадресацию электронной почты в Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
