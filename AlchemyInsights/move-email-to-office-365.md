# <a name="move-email-to-microsoft-365"></a>Перемещение электронной почты в Microsoft 365

У вас есть несколько вариантов переноса электронной почты в Microsoft 365. Консультант по миграции почты [Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) помогает выбрать оптимальный для вашей среды, а также направляет вас по миграции почты.
  
Чтобы ознакомиться с вариантами без запуска миграции, см. в материале ["Решение о пути миграции".](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path)

Проверьте также страницу [Принять Microsoft 365 на странице FastTrack,](https://www.microsoft.com/fasttrack/microsoft-365/office-365) чтобы получить помощь в миграции, если это имеет право.
