# <a name="how-to-add-a-user"></a>Добавление пользователя

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FOfN?autoplay=false]

1. В Центре администрирования Microsoft 365 выберите **Пользователи** > [Активные пользователи](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users).

2. Щелкните **Добавить пользователя**.

3. Заполните сведения о пользователе. После завершения нажмите кнопку **Завершить добавление**.

Дополнительные сведения см. в статье [Добавление пользователей по одному или массово](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users).
