# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Создание правил папки "Входящие" для общих почтовых ящиков

Можно добавить правила на общем почтовом ящике имеется доступ к подобным образом как добавление правил собственные учетной записи. 
  
Убедитесь, что являетесь участником интересующую общего почтового ящика и затем выполните действия, описанные в этой статье: [Добавление правил для общего почтового ящика](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164 )
