# <a name="how-to-type-special-characters-on-a-mac"></a>Ввод специальных символов на компьютерах Mac

Чтобы ввести специальные символы на компьютере Mac, откройте меню **специальные символы** с помощью сочетания клавиш ⌘ **командное** + **** + **пространство**управления.

быстрое сочетание клавиш для @ также также **ALT** + **64**.
