# <a name="proxyaddress-incorrect"></a>Неправильные ProxyAddress

Когда объект синхронизируется с Azure AD, значения, указанные в атрибуте proxyAddresses в Active Directory, сравниваются с правилами Azure AD, а затем атрибут proxyAddresses заполняется в Azure AD. Таким образом, значения атрибута proxyAddresses для объекта в Active Directory могут отличаться от значений атрибута proxyAddresses в Azure AD.
  
Чтобы узнать больше о заполнении proxyAddress, посмотрите, [как атрибут proxyaddress заполняется в Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
