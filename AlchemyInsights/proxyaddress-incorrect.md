# <a name="proxyaddress-incorrect"></a>Неправильный адрес

При синхронизации объекта для Azure AD значения, заданные в атрибуте proxyAddresses в Active Directory сравниваются с правилами Azure AD и затем заполнен атрибут proxyAddresses в Azure AD. Таким образом значения атрибута proxyAddresses для объекта в службе каталогов Active Directory могут не совпадать с значения атрибута proxyAddresses в Azure AD.
  
Чтобы узнать больше о том, как выполняется заполнение proxyaddress, обратитесь к разделу [как заполнен атрибут proxyAddress в Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
