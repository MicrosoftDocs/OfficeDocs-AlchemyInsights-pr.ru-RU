# <a name="disable-network-scan"></a>Отключение сканирования сети

Сканирование совместной сети может повлиять на производительность.  Чтобы клиент не сканирует сетевые акции/файлы по умолчанию, настройте следующие параметры в приложении Защитник Windows **True:**

- DisableScanningMappedNetworkDrivesForFullScan
- DisableScanningNetworkFiles
