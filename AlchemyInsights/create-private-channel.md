# <a name="create-a-private-channel"></a>Создание закрытого канала

По умолчанию любой владелец или член команды может создать закрытый канал. Гости не могут создавать закрытые каналы. 

Управление возможностью создания закрытых каналов может осуществляться на уровне команды и на уровне организации. Используйте политики, чтобы выбрать пользователей в организации, которым разрешено создавать закрытые каналы. Задав политики, владельцы команды могут включить или отключить привилегию создания закрытых каналов для участников на вкладке **Параметры**.

> [!NOTE]
> Каждая команда может иметь не более 30 закрытых каналов, а в каждом закрытом канале может быть не более 250 участников.

Дополнительные сведения см. в статье [Создание закрытого канала](https://docs.microsoft.com/MicrosoftTeams/private-channels#private-channel-creation).
