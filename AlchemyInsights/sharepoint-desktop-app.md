# <a name="desktop-app-for-sharepoint"></a>Классическое приложение для SharePoint

Ваша организация вводится в программе личной предварительной версии для настольного приложения для SharePoint.

В настоящее время корпорация Майкрософт не поддерживает классическое приложение для SharePoint (частный предварительный просмотр). Чтобы получить помощь в классическом приложении для SharePoint, используйте прикрепленных партнеров Microsoft Private Preview.
