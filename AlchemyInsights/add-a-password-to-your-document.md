# <a name="add-a-password-to-a-word-document"></a>Добавление пароля в документ Word

1. Перейдите к разделу "**сведения о** >  **файле** > "**защитить шифрование документа** > **с помощью пароля**.
2. Введите пароль, который вы хотите использовать, и нажмите кнопку **ОК**.
3. Повторно введите пароль, а затем нажмите кнопку **ОК**.
