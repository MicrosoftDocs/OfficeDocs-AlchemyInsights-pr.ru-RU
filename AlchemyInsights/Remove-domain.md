# <a name="trying-to-remove-your-domain"></a>Пытаетесь удалить домен?

Следуйте инструкциям из статьи [Удаление домена](https://docs.microsoft.com/office365/admin/get-help-with-domains/remove-a-domain).
  
Перед удалением домена необходимо обновить пользователей и группы, которые ему назначены. Другие стандартные решения см. в статье [Вопросы и ответы о доменах](https://docs.microsoft.com/office365/admin/setup/domains-faq).
