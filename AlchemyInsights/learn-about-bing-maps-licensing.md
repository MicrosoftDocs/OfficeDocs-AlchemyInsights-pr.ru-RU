# <a name="learn-about-bing-maps-licensing"></a>Узнайте о лицензировании Bing Maps

Служба Bing Maps предлагает несколько гибких вариантов лицензирования, которые будут поддерживать потребности и технические требования организации. Кроме того, с помощью бесплатного уровня вы можете начать строительство с помощью Bing Maps уже сегодня.

Подробные сведения [см. в материале Bing Maps Licensing](https://go.microsoft.com/fwlink/?linkid=2150203).
