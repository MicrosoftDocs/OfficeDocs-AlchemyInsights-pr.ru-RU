# <a name="learn-about-microsoft-365-groups"></a>Сведения о группах Microsoft 365

Группы Microsoft 365 — это служба, которая работает с уже используемыми инструментами Office 365, поэтому вы можете совместно работать с другими участниками при написании документов, создании электронных таблиц, работе с планами проектов, планировании собраний или отправке электронной почты. Более подробную информацию можно узнать в [статье сведения о группах Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
