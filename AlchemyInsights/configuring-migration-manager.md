# <a name="configuring-migration-manager"></a>Настройка диспетчера миграции

Диспетчер миграции централизует управление миграциями больших общих файловых ресурсов, настраивая один или несколько компьютеров или виртуальных машин (ВМ) в качестве «агентов» миграции. См. также:

- [Настройка клиентов диспетчера переноса данных](https://docs.microsoft.com/sharepointmigration/mm-setup-clients)

- [Параметры диспетчера переноса данных](https://docs.microsoft.com/sharepointmigration/mm-settings)
