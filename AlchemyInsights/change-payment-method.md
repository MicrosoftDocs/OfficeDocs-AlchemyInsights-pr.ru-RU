# <a name="change-payment-method"></a>Изменение метода оплаты

Метод оплаты можно изменить в **центре администрирования Microsoft 365**.
  
1. В центре администрирования выберите продукты для **выставления счетов** \> [& службы](https://go.microsoft.com/fwlink/p/?linkid=842054).

2. Найдите подписку, для которой требуется изменить способ оплаты. Рядом с пунктом метод оплаты нажмите кнопку **изменить**.

3. Следуйте инструкциям на экране, чтобы завершить изменение.

Чтобы узнать больше, ознакомьтесь со статьей [изменение способа оплаты для Microsoft 365 для бизнеса](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/change-payment-method).
