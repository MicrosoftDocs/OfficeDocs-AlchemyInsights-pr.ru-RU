# <a name="change-payment-method"></a>Изменение метода оплаты

Вы можете проверить способ оплаты, который настроен, перейдя на **выставление счетов** \> **подписки**. Выберите подписки, который нужно проверить. Способ оплаты, настроенный для подписки на отображается в разделе **способ оплаты**. Чтобы обновить сведения о методе существующих оплаты, нажмите кнопку **Обновить сведения об оплате**.
  
1. В Центре администрирования откройте страницу [Подписки](https://go.microsoft.com/fwlink/p/?linkid=842054) или выберите **Выставление счетов** \> **Подписки**.
    
2. Выберите подписку, которую требуется изменить способ оплаты для и нажмите кнопку **Изменить способ оплаты**.
    
3. Следуйте инструкциям на экране, чтобы завершить изменение.
    
Пошаговые инструкции, см.: [изменить способ платежа для Office 365 для бизнеса](https://support.office.com/article/8652f539-3123-4a8f-b9bd-6aa2f0e0372d)
