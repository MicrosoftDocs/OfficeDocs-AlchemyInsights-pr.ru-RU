# <a name="dashboard-not-loading"></a>Панель мониторинга не загружается

**Симптом:**

Панель мониторинга может зависнуть с вертушкой загрузки или отобразить общее сообщение об ошибке: "Возникла проблема. Подробности см. в технических сведениях".

**Руководство по устранению неполадок:**

См. статью [Панель мониторинга UCI не загружается после выпуска 2003.5](https://support.microsoft.com/help/4558635/uci-dashboard-not-loading-after-the-2003-5-release).
