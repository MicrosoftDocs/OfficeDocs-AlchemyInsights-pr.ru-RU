# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>Я не вижу ожидающих календарей при выборе доступности

Если вы не видите нужные календари, перейдите в раздел Календарь, нажмите кнопку меню в левом верхнем углу экрана и убедитесь, что выбраны нужные календари. Только календари, включенные в это меню, отображаются при выборе времени для обеспечения доступности при отправке.
