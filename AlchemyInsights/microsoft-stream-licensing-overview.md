# <a name="microsoft-stream-licensing-overview"></a>Общие сведения о лицензировании Microsoft Stream

Microsoft Stream — это облачная служба, лицензированная на основе подписок на пользователя. Он доступен в рамках соответствующей подписки на Office 365. В статье Office 365 ценообразование представлены сведения о каждом типе подписки.
