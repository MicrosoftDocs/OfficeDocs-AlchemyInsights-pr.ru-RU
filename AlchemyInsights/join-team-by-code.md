# <a name="join-a-team-by-code"></a>Присоединение к группе с помощью кода

Корпорации Майкрософт известно о проблеме, из-за которой пользователям не удается создать код **Присоединиться к группе**, или присоединиться к группе с помощью функции **Присоединиться с помощью кода**. В качестве обходного решения владельцы групп могут вручную добавить пользователей, если это потребуется.
