2. <span data-ttu-id="fdc64-p102">**Внешние пользователи в группе внешние** — внешних групп разрешить внешним пользователям. Области доступа внешних пользователей ограничен внешней группы. С помощью внешних групп только можно приглашать контакты из сети Yammer, позволяющие пользователям принимать участие в беседах по внешним.</span><span class="sxs-lookup"><span data-stu-id="fdc64-p102">**External users in an external group** - External groups allow external users. 3. <span data-ttu-id="fdc64-p103">**Внешние пользователи в сети Yammer беседу в внутренней группы** — этот параметр позволяет внешним участникам для добавления к определенной беседы. Внешние участники только могут принимать участие в беседах, которые они приглашены явно для и вложений в беседу. Они получают доступ к эти беседы с помощью Yammer папки входящих сообщений (в их сети). Они не имеют доступа к остальной части сети.</span><span class="sxs-lookup"><span data-stu-id="fdc64-p103">**External users in a Yammer conversation in an internal group** - This option allows external participants to be added to a particular conversation. External participants can only participate in conversations they have been explicitly invited to and to any attachments in that conversation. They access these conversations via their Yammer inbox (on their network). They have no access to the rest of your network.</span></span> 
    
4. <span data-ttu-id="fdc64-p104">**Внешние сети** — этот параметр, можно приглашать внешних пользователей к внешней сети, проверена, администраторы могут создавать в вашей сети.
