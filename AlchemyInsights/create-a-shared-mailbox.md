# <a name="create-a-shared-mailbox"></a>Создание общего почтового ящика

Общий почтовый ящик позволяет нескольким пользователям отправлять и получать электронную почту для псевдонима, например **Sales@contoso.com**, без назначения другой лицензии.
  
Для получения дополнительных сведений ознакомьтесь со статьей [Создание общего почтового ящика](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) .
