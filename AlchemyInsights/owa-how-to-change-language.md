# <a name="change-language"></a>Изменение языка

1.    Войдите в Outlook в Интернете.
2. В верхней части страницы выберите **Параметры** > **Просмотр всех параметров Outlook**.
3. Выберите **Общий** > **язык и время**.
4. Выберите язык, который вы хотите использовать.
5. Нажмите кнопку **Сохранить**.
 
Дополнительные сведения см в разделе [Изменение часового пояса и языковых параметров в Outlook в Интернете](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
