# <a name="turn-off-focused-inbox"></a>Отключить фокус папки "Входящие"

1. Выберите пункт **Параметры**![параметры](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) \> **фокус папки «Входящие»**.
    
2. В разделе **при получении электронной почты**выберите **не сортировки сообщений**.
    
3. Нажмите кнопку **ОК**.
    
Для получения дополнительных сведений см [Фокус папки "Входящие" для Outlook](https://go.microsoft.com/fwlink/p/?linkid=873108).
