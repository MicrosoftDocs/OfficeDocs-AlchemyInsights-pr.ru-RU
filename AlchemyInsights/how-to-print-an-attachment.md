# <a name="print-an-attachment"></a>Печать вложения

Чтобы напечатать вложения электронной почты, скачайте и сохраните вложения на локальном компьютере или OneDrive, а затем откройте их с помощью соответствующей программы.
  
1. Откройте сообщение, содержащее вложение, которое требуется напечатать.
    
2. Под вложением нажмите кнопку **загрузить**. 
    
3. После загрузки файла откройте соответствующую программу и выберите параметр для печати. Например, откройте файл doc в Word, а затем выполните печать.
    
Для получения дополнительных сведений прочитайте статью [Печать сообщений электронной почты и вложений в Outlook.com](https://go.microsoft.com/fwlink/?linkid=2021110&amp;clcid=0x409).
