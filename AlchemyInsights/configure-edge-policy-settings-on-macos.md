# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Использование Jamf для настройки параметров политики Microsoft Edge на macOS

Чтобы настроить Microsoft Edge (стабильный канал 81 или более поздний) на macOS, используйте файл манифеста политики Microsoft Edge в Jamf Pro 10.19.

Дополнительные сведения см. в [перенастройке параметров](https://go.microsoft.com/fwlink/?linkid=2134761)политики Microsoft Edge на macOS с помощью Jamf.
