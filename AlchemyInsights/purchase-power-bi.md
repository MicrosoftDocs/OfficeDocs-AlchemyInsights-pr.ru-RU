# <a name="purchase-power-bi"></a>Приобретение Power BI

Power BI можно приобрести в **центре администрирования Microsoft 365**.

1. Перейдите на страницу **Услуги по выставлению счетов > [приобретений](https://go.microsoft.com/fwlink/p/?linkid=868433) ** .

2. Используйте поле поиска для поиска по запросу "Power BI".

3. Выберите план Power BI, который вы хотите приобрести.

4. Выберите количество пользователей, которые будут использовать Power BI.

Чтобы назначить лицензии пользователям, ознакомьтесь со статьей [Назначение лицензий пользователям](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide).

Сведения о приобретении и удалении лицензий см. в статье [Добавление и удаление лицензий для подписок](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription).
