# <a name="ignorelegalhold-switch"></a>Параметр **–IgnoreLegalhold**

Параметр **–IgnoreLegalhold**, используемый в следующем командлете, недоступен и не поддерживается для пользовательских объектов Exchange Online.

- Disable-mailbox
- Remove-mailbox
- Disable-mailuser
- Remove-mailuser
