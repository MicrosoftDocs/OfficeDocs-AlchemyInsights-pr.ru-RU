# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>Обновление DNS-записей для сохранения веб-сайта у текущего поставщика услуг размещения веб-сайтов

В рамках процесса перемещения общедоступного веб-сайта Microsoft 365 на другого поставщика услуг хостинга, скорее всего, потребуется внести изменения в записи домена.
  
1. В центре администрирования перейдите в раздел **Настройка** \> **доменов** и выберите доменное имя, которое требуется изменить.

2. Введите сведения, соответствующие вашему веб-сайту, с помощью нового поставщика услуг хостинга.

Чтобы узнать больше, ознакомьтесь [со статьей обновление DNS-записей, чтобы сохранить текущий поставщик услуг хостинга на веб-сайте](https://docs.microsoft.com/microsoft-365/admin/dns/update-dns-records-to-retain-current-hosting-provide).
  
Чтобы узнать больше о прекращении работы общедоступных сайтов, просмотрите [общедоступные веб-сайты SharePoint Online](https://support.office.com/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916).
