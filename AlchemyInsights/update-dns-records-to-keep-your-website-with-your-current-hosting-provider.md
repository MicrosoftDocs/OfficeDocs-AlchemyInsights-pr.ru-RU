# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>Обновление DNS-записей для сохранения веб-сайта у текущего поставщика услуг размещения веб-сайтов

В процессе перехода на Office 365 общедоступного веб-сайта для другого поставщика услуг размещения необходимо внести изменения в записи домена.
  
1. В центре администрирования, перейдите в окно **Установка** \> **домены** и выберите имя домена, который требуется изменить. 
    
2. Введите сведения, посвященные на веб-сайт нового поставщика услуг размещения.
    
Для получения дополнительных сведений см [обновления DNS-записей для хранения веб-сайта у текущего поставщика услуг размещения](https://support.office.com/en-us/article/update-dns-records-to-keep-your-website-with-your-current-hosting-provider-2c4cf347-b897-45c1-a71f-210bdc8f1061). 
  
Чтобы узнать больше о прекращении поддержки общедоступного сайта видеть [SharePoint Online общедоступных веб-сайтов поддержки](https://support.office.com/en-us/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916?ui=en-US&amp;rs=en-US&amp;ad=US).
