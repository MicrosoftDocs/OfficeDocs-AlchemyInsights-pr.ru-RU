# <a name="microsoft-365-multi-geo"></a>Microsoft 365 с поддержкой нескольких регионов

- [Microsoft 365 с поддержкой нескольких регионов](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)

- [Поддержка нескольких регионов в Exchange Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-exchange-online)
- [Планирование для Microsoft 365 с поддержкой нескольких регионов](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Настройка поддержки нескольких регионов](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
