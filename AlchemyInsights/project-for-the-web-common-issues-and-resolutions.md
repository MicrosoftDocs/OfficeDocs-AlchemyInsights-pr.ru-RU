# <a name="project-for-the-web-common-issues-and-resolutions"></a>Распространенные проблемы Project в Интернете и их решение

Ниже приведены некоторые актуальные темы, касающиеся Project в Интернете.

- Сведения о [Project в Интернете](https://support.microsoft.com/ru-RU/office/what-is-project-for-the-web-c19b2421-3c9d-4037-97c6-f66b6e1d2eb5), доступном во всех [трех подписках Project](https://products.office.com/project/compare-microsoft-project-management-software).
- Отключение или включение Project в Интернете или Дорожной карты [для пользователей организации](https://docs.microsoft.com/project-for-the-web/turn-project-for-the-web-off). Администраторы могут управлять доступом к Project в Интернете с помощью страницы **Параметры Project** в Центре администрирования Майкрософт.
- Узнайте больше о [возможностях](https://support.office.com/article/what-can-you-do-with-project-for-the-web-b30f5442-be5f-43d2-9072-c95bff778ea1) Project в Интернете.
- Помогите пользователям создавать проекты, используя [учебные материалы по Project в Интернете](https://support.office.com/article/get-started-with-project-for-the-web-50bf3e29-0f0d-4b7a-9d2c-7c78389b67ad).
- Ознакомьтесь с [ограничениями и границами](https://docs.microsoft.com/project-for-the-web/project-for-the-web-limits-and-boundaries) Project в Интернете.

Дополнительные сведения о Project в Интернете см. в разделе [Справка для администраторов Project в Интернете](https://docs.microsoft.com/project-for-the-web/projectforweb-admin-home).
