# <a name="sharing-advanced-hunting-queries"></a>Совместное использование расширенных запросов на поиск

Расширенные запросы на поиск могут совместно использоваться между пользователями одной организации. Вы также можете найти общедоступные запросы в GitHub. Эти запросы позволяют быстро реализовывать сценарии охоты на угрозы, не создавая запросы с нуля.
  
Дополнительные сведения см [в разделе Использование общих запросов в расширенном поиске](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/advanced-hunting-shared-queries).
