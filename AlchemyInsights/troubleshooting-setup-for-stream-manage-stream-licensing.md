# <a name="managing-microsoft-stream-licenses"></a>Управление лицензиями Microsoft Stream

**Важно!** Для выполнения операций, описанных в этом разделе, необходимо быть членом роли администратора клиента.

Необходимо быть глобальным администратором или администратором управления пользователями. Более подробную информацию можно узнать в статье [сведения о ролях администратора](https://docs.microsoft.com/microsoft-365/admin/add-users/about-admin-roles).

Вы можете назначить [лицензии учетным записям пользователей с помощью Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
