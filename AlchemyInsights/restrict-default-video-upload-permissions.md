# <a name="restrict-default-video-upload-permissions"></a>Ограничение разрешений на отправку видео по умолчанию

Когда пользователи отправляют видео в Microsoft Stream с помощью потокового портала вне контекста группы или канала группы, флажок "разрешить всем пользователям в организации просматривать это видео" установлен по умолчанию. Это позволит просматривать видео всем сотрудникам вашей компании.

- [Настройка разрешений на отправку видео по умолчанию](https://docs.microsoft.com/stream/default-video-permissions)
