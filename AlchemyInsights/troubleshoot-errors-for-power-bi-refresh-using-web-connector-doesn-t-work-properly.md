# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Обновление с использованием Web Connector не работает должным образом

Если у вас есть скрипт веб-соединителя, использующий функцию [Web. Page](https://msdn.microsoft.com/library/mt260924.aspx) , и вы обновили набор данных или отчет после 18 ноября 2016, то вам потребуется использовать шлюз для правильной работы обновления.

Дополнительные сведения:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
