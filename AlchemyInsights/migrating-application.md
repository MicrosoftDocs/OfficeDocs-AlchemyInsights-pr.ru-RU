# <a name="migrating-applications"></a>Перенос приложений

[Ресурсы для переноса](https://docs.microsoft.com/azure/active-directory/manage-apps/migration-resources) приложений в Azure Active Directory — ресурсы, которые помогут перенести доступ к приложениям и проверку подлинности в Azure Active Directory (Azure AD).

[Перемещение проверки](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) подлинности приложений из служб федерации Active Directory в Azure Active Directory — руководство разработчика по переносу приложений из служб федерации Active Directory в Azure Active Directory.
