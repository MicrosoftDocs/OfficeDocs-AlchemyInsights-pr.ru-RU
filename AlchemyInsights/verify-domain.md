# <a name="trying-to-verify-your-domain"></a>При попытке проверки домена?

Необходимо добавить запись TXT для вашего регистратора доменных имен, чтобы убедиться, что вы владеете вашего домена. Следуйте указаниям мастера добавления доменов, нажав кнопку **Параметры** \> **домена**, а затем **Добавить домен**. 
  
Следуйте инструкциям, представленным в статье под для добавления записи TXT для проверки вашего домена. Выберите регистратора доменных имен конкретные инструкции для вашего домена.
  
[Создание записей для Office 365](https://support.office.com/article/https://support.office.com/article/Create-DNS-records-for-Office-365-when-you-manage-your-DNS-records-B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23.aspx)
