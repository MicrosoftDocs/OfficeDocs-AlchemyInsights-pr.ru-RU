# <a name="solutions-for-unsupported-operating-system"></a>Решения для "неподдерживаемая операционная система"

При попытке установить Microsoft 365 может появиться сообщение об ошибке **неподдерживаемой операционной системы** , если устройство не соответствует [требованиям к системе](https://products.office.com/office-system-requirements).
  
Для Microsoft 365 требуется:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 с пакетом обновления 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 или Windows Server 2008 R2

    OR

- Mac OS X 10.10

Если вы получили это сообщение при попытке установить Office на мобильном устройстве или Хромбук, ознакомьтесь с [инструкциями по установке и запуску Microsoft Office на Хромбук](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
