# <a name="solutions-for-unsupported-operating-system"></a>Решения для «Неподдерживаемый операционной системы»



При попытке установить Office 365, если ваше устройство не соответствует [требования к системе](https://products.office.com/office-system-requirements)может появиться сообщение об ошибке **не поддерживается операционной системы** .
  
Office 365 требуется:
  
- ПК: Windows 10, Windows 8.1, Windows 8, Windows 7 с пакетом обновления 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 или Windows Server 2008 R2
    
    ИЛИ
    
- Mac OS X 10.10
    
Если получено сообщение при попытке установить Office на мобильное устройство или Chromebook, узнайте, [как установить и запустить Microsoft Office на Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
