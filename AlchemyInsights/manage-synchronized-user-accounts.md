# <a name="manage-synchronized-user-accounts"></a>Управление учетными записями синхронизированных пользователей

Как правило используйте Центр администрирования Office 365 для добавления и изменения учетных записей пользователей. Тем не менее его нельзя использовать, если включена синхронизации каталогов или службу единого входа. Вместо этого управления пользователями с помощью Windows PowerShell, как описано в разделе [Настройка свойств учетной записи пользователя с Office 365 PowerShell](https://docs.microsoft.com/en-us/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
