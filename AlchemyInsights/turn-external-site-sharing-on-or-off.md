# <a name="turn-external-sharing-on-or-off-for-a-site"></a>Включение и отключение внешнего общего доступа для сайта

Для изменения параметров внешнего общего доступа для сайта (ранее называемого "семейством веб-сайтов") необходимо быть глобальным администратором или администратором SharePoint. Владельцы сайта не могут изменить этот параметр. 

Дополнительную информацию можно узнать [в статье Включение и отключение внешнего общего доступа для сайта](https://docs.microsoft.com/sharepoint/change-external-sharing-site).
