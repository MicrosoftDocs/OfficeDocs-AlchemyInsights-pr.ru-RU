# <a name="issue-with-skype-for-business"></a>Проблемы, связанной с Скайп для бизнеса?

Мы обнаружила, что следующие домены дает проблем, которые могут запретить работают должным образом Скайп для бизнеса:
  
{HealthCheckString}
  
Записи NS, CNAME или SRV параметров домена (DNS) не настроена правильно для Скайп для бизнеса. Дополнительные сведения можно найти [обновления DNS-записей в Office 365](https://support.office.com/article/Create-DNS-records-for-Office-365-when-you-manage-your-DNS-records-B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23.aspx) . 
  
Чтобы устранить ее, перейдите на страницу " [домены](https://admin.microsoft.com/adminportal/home#/Domains) " в центре администрирования Office 365 и поможет с ним. 
  
Если возникают ошибки при взаимодействии с пользователями, которые являются внешними по отношению к вашей организации, можно найти [Set up Скайп для бизнеса в Интернет внешних коммуникаций](https://support.microsoft.com/help/10041/set-up-skype-for-business-online-external-communications.aspx) подробные инструкции. 
  
[Проверьте вопросы и ответы по домены](https://support.office.com/article/7b7b075d-79f9-4e37-8a9e-fb60c1d95166.aspx) другие распространенные решения.
