# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Ошибка при открытии общего почтового ящика в Outlook в Интернете?

Если возникнет ошибка, **что-то пошло не так. У вас нет разрешения на открытие этого почтового ящика.** При попытке открыть общий почтовый ящик в Outlook в Интернете у вашей учетной записи может отсутствовать разрешение на доступ к общему почтовому ящику. Администратор организации может назначить разрешения учетной записи для общего почтового ящика, следуя инструкциям, приведенным в разделе [Создание общего почтового ящика](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).
