# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Ошибка при открытии общего почтового ящика в Outlook в Интернете?

Если возникнет ошибка, **что-то пошло не так. У вас нет разрешения на открытие этого почтового ящика.** При попытке открыть общий почтовый ящик в Outlook в Интернете у вашей учетной записи может отсутствовать разрешение на доступ к общему почтовому ящику. Администратор организации может назначить разрешения учетной записи для общего почтового ящика, следуя инструкциям, приведенным в разделе [Создание общего почтового ящика](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox).
