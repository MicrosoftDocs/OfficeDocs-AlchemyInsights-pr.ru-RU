# <a name="block-pop-up-windows-in-microsoft-edge"></a>Блокировка всплывающих окон в Microsoft Edge

1. Перейти к **... (Параметры и другие)**  >  **Параметры**  >  **Разрешения для сайта**.
2. Выберите пункт **всплывающие окна и перенаправление**.
3. Включение **блока**.
