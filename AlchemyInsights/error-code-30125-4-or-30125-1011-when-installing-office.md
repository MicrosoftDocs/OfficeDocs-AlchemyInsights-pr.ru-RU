# <a name="solutions-for-error-30125-4-or-30125-1011"></a>Решения для ошибок 30125-4 или 30125 1011

Эта ошибка может появиться, если антивирусное программное обеспечение, брандмауэра, параметров прокси-сервера или подключения не позволяют установки Office. Ниже приведены некоторые факторы, можно попробовать решить проблему.
  
- Перезагрузите компьютер и повторно установить Office
    
- Установка Office с помощью [автономной установки Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c.aspx)
    
- Временно отключить параметры прокси-сервера
    
- Временное отключение антивирусной программы
    
- Временное отключение брандмауэра
    
Для получения дополнительных сведений см. [код ошибки 30125-4 или 30125 1011 при установке Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612.aspx)
