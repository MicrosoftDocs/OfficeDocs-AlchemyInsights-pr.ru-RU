# <a name="does-visio-work-on-a-mac"></a>Работает ли Visio в Mac?

Для Mac не существует приложения для настольного компьютера Visio. Тем не менее, если пользователю, которому требуется приложение, назначена [Лицензия Visio Online](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799), пользователи могут [просматривать, создавать и редактировать схемы в Visio Online](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) в веб-браузере.
