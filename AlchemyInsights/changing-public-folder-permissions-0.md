# <a name="changing-public-folder-permissions"></a>Изменение разрешений для общей папки

Разрешения общей папки можно изменить пользователями и администраторами в Outlook. Администраторы также могут управлять разрешения из Exchange центр Администрирования, выполнив следующие:
  
1. В портале O365 с помощью левого меню переходов, перейдите к разделу **Центры администрирования** \> **Exchange**.
    
2. Перейдите к разделу **общедоступные папки**.
    
3. Отсюда можно изменить разрешения для отдельных общих папок, назначение группы безопасности для разрешений. Пользователь может изменить разрешениями для общих папок пользователю необходимо иметь права владельца в общей папке.
