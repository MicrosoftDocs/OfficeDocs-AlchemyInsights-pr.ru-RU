# <a name="how-to-delete-a-folder"></a>Удаление папки

1. В области папок щелкните правой кнопкой мыши папку или подпапку, которую нужно удалить.
2. Выберите пункт **Удалить папку**.
3. Нажмите кнопку **ОК** , чтобы подтвердить.

Чтобы узнать больше, ознакомьтесь [со статьей работа с папками сообщений в Outlook в Интернете](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
