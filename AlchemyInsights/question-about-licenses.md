# <a name="question-about-licenses"></a>Вопрос о лицензиях

Если вы хотите узнать, требуемое количество лицензий, вы приобрели или сколько лицензий назначенных по подписке:
  
1. В Центре администрирования откройте страницу [Подписки](https://go.microsoft.com/fwlink/p/?linkid=842054) или выберите **Выставление счетов** \> **Подписки**.
    
2. Выберите подписку, чтобы получить дополнительные сведения.
    
3. В левой части карточки подписки, сведения о лицензии:
    
  - Номер **Куплено** — число лицензий, которые вы оплаты. 
    
  - **Назначенный** номер — число лицензий назначенных пользователю. 
    
4. Если вы хотите добавить дополнительные лицензии, есть ли у вас есть слишком многих, нажмите кнопку **Добавить или удалить лицензии** для настройки приобрести отдельно.
