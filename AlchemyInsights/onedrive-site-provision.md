# <a name="set-up-onedrive"></a>Настройка OneDrive

Для новых учетных записей пользователям необходимо войти на портал Office 365 с собственной учетной записью и щелкнуть значок OneDrive, чтобы вручную запустить подготовку сайта OneDrive.

Тем не менее, если пользователь уже выбрал значок OneDrive, а его личный сайт не подготовлен в течение 24 часов, запишите билет поддержки для получения дополнительной помощи.
