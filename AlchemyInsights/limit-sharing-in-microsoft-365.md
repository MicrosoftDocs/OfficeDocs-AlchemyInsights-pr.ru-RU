# <a name="limit-sharing-in-microsoft-365"></a>Ограничение общего доступа в Microsoft 365

Хотя полностью отключить внутренний общий доступ или удалить кнопку Share с сайтов нельзя, существует множество способов ограничить общий доступ в Microsoft 365 для удовлетворения потребностей организации. Дополнительные сведения см. в [разделе Limit sharing in Microsoft 365.](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)
