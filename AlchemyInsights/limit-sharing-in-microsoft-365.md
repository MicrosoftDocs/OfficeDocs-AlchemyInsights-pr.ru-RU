# <a name="limit-sharing-in-microsoft-365"></a>Ограничение общего доступа в Microsoft 365

Хотя вы не можете полностью отключить внутренний общий доступ или удалить кнопку Общий доступ с сайтов, существует множество способов ограничения общего доступа в Microsoft 365 для удовлетворения потребностей Организации. Для получения дополнительных сведений ознакомьтесь [с разделом ограничить общий доступ в Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
