# <a name="getting-an-exchange-hybrid-key"></a>Извлечение гибридного ключа Exchange

Чтобы подключить локальную организацию к Exchange Online в Office 365, необходимо настроить по крайней мере один сервер Exchange в локальной организации. Этот сервер используется для создания гибридного развертывания, которое подключает локальную организацию Exchange в Exchange Online. Чтобы получить гибридный ключ для этого сервера, выполните действия, описанные в [мастере распространения ключей гибридного ключа продукта Exchange](https://aka.ms/hybridkey).
