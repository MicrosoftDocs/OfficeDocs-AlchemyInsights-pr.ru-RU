# <a name="getting-an-exchange-hybrid-key"></a>Получение ключа гибридного Exchange

Для подключения локальной организации Exchange Online в Office 365, необходимо настроить хотя бы один сервер Exchange server в локальной организации. Этот сервер используется для создания гибридного развертывания, который подключается к локальной организации Exchange в Exchange Online. Получить ключ гибридного для этого сервера, выполните действия в [Мастер распространения ключей продукта гибридного Exchange](http://aka.ms/hybridkey).
