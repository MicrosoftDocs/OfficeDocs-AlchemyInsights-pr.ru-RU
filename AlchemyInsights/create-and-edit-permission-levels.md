# <a name="working-with-sharepoint-permission-levels"></a>Работа с уровнями разрешений SharePoint

SharePoint поставляется с уровнями разрешений по умолчанию, которые можно **использовать,** например **Contribute** и View только . Но если один из них не отвечает вашим потребностям, можно создать новый уровень разрешений. Имейте в виду, что легче отслеживать уровни разрешений, если вы не измените по умолчанию.

[Создание и редактирование уровней разрешений](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
