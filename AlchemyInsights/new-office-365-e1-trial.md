# <a name="new-office-365-e1-trial"></a>Новая пробная версия Office 365 E1

По истечение 1 июля 2020 пробная лицензия Office 365 E1 больше недоступна. Если вам нужно лицензировать пользователей для Microsoft Teams, ознакомьтесь с [описанием службы Microsoft Teams](https://docs.microsoft.com/office365/servicedescriptions/teams-service-description) для получения списка платных подписок, включающих Teams. Кроме того, подходящие организации могут использовать [бесплатную версию Teams](https://support.office.com/article/Welcome-to-Microsoft-Teams-free-6d79a648-6913-4696-9237-ed13de64ae3c)или сотрудники могут активировать [произвольную работу Teams](https://docs.microsoft.com/MicrosoftTeams/teams-exploratory) .
