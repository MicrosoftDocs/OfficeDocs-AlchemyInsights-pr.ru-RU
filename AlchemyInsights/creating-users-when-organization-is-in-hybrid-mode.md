# <a name="create-users-in-hybrid-deployments"></a>Создание пользователей в гибридных развертываниях

Если ваша организация является гибридным развертыванием между локальной средой и Microsoft 365, необходимо создать все объекты пользователя в локальной службе Active Directory. Вы создадите объекты пользователя в соответствующем подразделении (OU), чтобы они синхронизировались в вашей организации.
