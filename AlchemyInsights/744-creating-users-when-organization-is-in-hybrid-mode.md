# <a name="create-users-in-hybrid-deployments"></a>Создание пользователей в гибридных развертываниях

Если ваша организация гибридного развертывания между локальной средой и Office 365, необходимо создать всех объектов-пользователей в Active Directory в локальной. Объектов пользователей создайте в соответствующие подразделения (OU), поэтому они в случае синхронизируются в организации Office 365.
