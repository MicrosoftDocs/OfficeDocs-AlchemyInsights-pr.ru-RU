# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Использование условий или расширенных формул в последовательности

В Microsoft Flow можно настроить процесс, чтобы он выполнял задачи только в том случае, если условие имеет значение true. Сведения см. в разделе [Добавление условия в последовательность](https://go.microsoft.com/fwlink/?linkid=872112).
  
Чтобы создать расширенную формулу, ознакомьтесь со статьей [схема языка определения рабочих процессов для приложений логики Azure](https://aka.ms/logicexpressions).
