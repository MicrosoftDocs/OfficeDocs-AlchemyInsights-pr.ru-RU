# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Использование условий или расширенных формул в потоке

Можно настроить поток, чтобы оно выполняет задачи, только в том случае, если условие равно true. Информация в разделе [Добавить условие к потоку](https://go.microsoft.com/fwlink/?linkid=872112).
  
Чтобы создать дополнительные формулы, видеть [схемы языка определения рабочего процесса для Azure логики приложения](https://aka.ms/logicexpressions).
