# <a name="stream-quality"></a>Качество Stream

В Stream используется одноадресная потоковая передача с переменной скоростью (ABR) HTTPS, чтобы динамически выбирать наилучшее качество воспроизведения видео с учетом доступной пропускной способности сети и размера видеопроигрывателя. Дополнительные сведения см. в статье:

- [Общие сведения о доставке видео и сети Microsoft Stream](https://docs.microsoft.com/stream/network-overview)
