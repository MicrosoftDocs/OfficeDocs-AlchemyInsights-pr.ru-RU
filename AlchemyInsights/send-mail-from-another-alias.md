# <a name="send-email-from-an-alias-or-secondary-address"></a>Отправлять электронную почту из псевдоним или дополнительный адрес

Office 365 в настоящее время не поддерживает отправку электронной почты из псевдоним или адрес электронной почты получателя. Псевдонимы электронной почты можно использовать для получения только по электронной почте. Если вы хотите отправлять электронную почту из другой адрес электронной почты, необходимо создать общий почтовый ящик. Общего почтового ящика можно получить другой адрес электронной почты, не требуя другой лицензии. 
  
- [Для создания общего почтового ящика, перейдите сюда](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)
    
Можно также ознакомьтесь со статьей [Создание общего почтового ящика](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9) , который сообщит вам, все, что вам нужно знать об общих почтовых ящиков, в том числе об их использовании.
