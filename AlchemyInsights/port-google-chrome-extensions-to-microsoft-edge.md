# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Port Extensions Google Chrome to Microsoft EDGE (Чромиум)

Вы можете легко [перенести расширения Google Chrome в Microsoft EDGE (чромиум)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). В большинстве случаев для запуска этих расширений в Microsoft Edge необходимы только минимальные изменения.

API расширения и ключи манифестов, поддерживаемые Google Chrome, совместимы с кодом Microsoft Edge. Однако Microsoft EDGE не поддерживает API расширения Chrome. gcm, Chrome. Identity. authz, Chrome. Identity. Жетаустокен и Chrome. instanceID.
