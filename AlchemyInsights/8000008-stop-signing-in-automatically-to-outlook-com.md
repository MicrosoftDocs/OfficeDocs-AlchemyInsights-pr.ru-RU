# <a name="how-to-stop-signing-in-automatically"></a>Как остановить автоматический вход

Чтобы остановить автоматический вход, снимите флажок для **оставаться в системе** на страницы, где можно вводить пароль. 
  
Для получения дополнительных сведений, узнайте, [как выполнить вход в группы или из Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873113).
