# <a name="skype-icon-missing"></a>Отсутствует значок Skype

Мы временно удалили значок Skype из Outlook.com в составе обновления и обновления компонентов, которые мы очень рады выполнить развертывание. Мы ценим ваше терпение, пока мы подготавливаем новый и Улучшенный интерфейс.

Вы по-прежнему можете использовать Skype через браузер по адресу [Web.Skype.com](https://web.skype.com).
