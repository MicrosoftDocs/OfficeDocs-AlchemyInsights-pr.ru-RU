# <a name="create-and-use-a-shared-mailbox"></a>Создание и использование общего почтового ящика

Общий почтовый ящик позволяет нескольким пользователям отправлять и получать электронную почту для псевдонима, например **Sales@contoso.com**, без назначения другой лицензии.
  
[Создание общего почтового ящика](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) сведения о том, как: 
  
- Создание общего почтового ящика
    
- Предоставление пользователям разрешений на использование общего почтового ящика
    
- Доступ к общему почтовому ящику в Outlook или Outlook в Интернете
