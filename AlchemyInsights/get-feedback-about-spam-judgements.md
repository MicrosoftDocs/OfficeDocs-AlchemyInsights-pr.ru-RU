# <a name="get-feedback-about-spam-judgments"></a>Получение отзывов о нежелательной почте

Администратор может легко сообщить о подозрительных сообщениях электронной почты, URL-адресах и содержимом в корпорацию Майкрософт в [обозревателе с отправкой](https://protection.office.com/reportsubmission). Вы даже можете использовать его, чтобы определить, почему законное сообщение было отмечено неправильно. Для получения дополнительных сведений обратитесь к этому [уведомлению](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

Для облегчения ИТ можно разрешить пользователям отправлять отчеты непосредственно в корпорацию Майкрософт с помощью [надстройки Report Message](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
