# <a name="add-seats-to-a-prepaid-subscription"></a>Добавление рабочих мест в будущих подписки

1. В Центре администрирования откройте страницу [Подписки](https://go.microsoft.com/fwlink/p/?linkid=842054) или выберите **Выставление счетов** \> **Подписки**.
    
2. Выберите подписку на будущих, которую вы хотите добавить пользователей.
    
3. В меню **действия** выберите пункт **обновления или добавить лицензию с маркером**.
    
4. Выберите параметр **ключ продукта новые и неиспользуемыми** . (Если этой подписки был создан с помощью ключа продукта, приобрести в розничном месте, может иметь параметр, чтобы добавить пользователей с помощью других способов оплаты.) 
    
5. Нажмите кнопку **Next** (Далее).
    
6. Введите ключ продукта и нажмите кнопку **Далее**.
    
7. Выберите параметр, чтобы добавить несколько пользователей, нажмите кнопку **Далее**, а затем выберите **Redeem**.
