# <a name="add-seats-to-a-prepaid-subscription"></a>Добавление мест к предоплаченной подписке

1. В Центре администрирования откройте страницу [Подписки](https://go.microsoft.com/fwlink/p/?linkid=842054) или выберите **Выставление счетов** \> **Подписки**.

2. Выберите предоплаченную подписку, которую вы хотите добавить для пользователей.

3. В меню **More Actions (дополнительные действия** ) выберите **Обновить или добавить лицензию с маркером**.

4. Выберите параметр **новый и неиспользованный ключ продукта** . (Если эта подписка была создана с помощью ключа продукта, приобретенного в розничном расположении, вы можете добавить пользователей с другими способами оплаты).

5. Нажмите кнопку **Далее**.

6. Введите ключ продукта и нажмите кнопку **Далее**.

7. Выберите параметр для добавления других пользователей, нажмите кнопку **Далее**, а затем выберите команду **активировать**.
