# <a name="microsoft-threat-experts"></a>Microsoft Threat Experts

Microsoft Threat Experts — это служба управляемой охоты на угрозы в Microsoft Defender для конечной точки.  Мы сотрудничаем с Вашим Центром операций безопасности для мониторинга, анализа и поддержки для выявления критических угроз в уникальной среде и реагирования на них.

Эта служба управляемой охоты на угрозы предоставляет экспертные сведения и данные с помощью этих двух возможностей: целевого уведомления об атаке и доступа к экспертам по запросу. Подробные сведения см. в документации [экспертов](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-threat-experts) по угрозам Майкрософт.
