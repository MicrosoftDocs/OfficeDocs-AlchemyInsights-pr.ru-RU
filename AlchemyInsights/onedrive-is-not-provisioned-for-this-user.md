# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive не подготовлен для этого пользователя

Для новых учетных записей пользователям необходимо войти на портал Office 365 с собственной учетной записью и выбрать OneDrive, чтобы вручную запустить подготовку сайта OneDrive.
  
Тем не менее, если пользователь уже выбрал OneDrive и его личный сайт не подготовлен в течение 24 часов, запишите билет поддержки для получения дополнительной помощи.
