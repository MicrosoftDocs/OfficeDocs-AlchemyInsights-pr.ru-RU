# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Изменение параметров языка интерфейса в SharePoint или OneDrive 

Изменение языковых параметров по умолчанию для SharePoint или OneDrive невозможно, так как эти параметры были определены при создании подписки на Microsoft 365. Однако вы можете изменить параметры языка интерфейса.

Чтобы изменить персональные параметры, ознакомьтесь со статьей  [изменение личных параметров языка и региона](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Чтобы изменить пользовательский интерфейс сайта, ознакомьтесь [со статьей выбор языков, которые должны быть доступны для пользовательского интерфейса сайта](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Дополнительные сведения об устранении неполадок в языковых параметрах см. [невозможно изменить язык по умолчанию для семейства веб-сайтов SharePoint Online](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).
