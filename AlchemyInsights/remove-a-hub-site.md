# <a name="remove-a-hub-site"></a>Удаление центрального сайта

Отмена регистрации центрального сайта в новом центре администрирования SharePoint
  
1. В новом центре администрирования SharePoint в разделе **сайты**щелкните **активные сайты**.

2. Выберите сайт, щелкните **центральный сайт**, а затем выберите пункт **отменить регистрацию в качестве центрального сайта**.

3. Нажмите кнопку **ОК**.

[Дополнительные сведения о центральных сайтах](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
