# <a name="remove-a-hub-site"></a>Удалять через центральный сайт

Отмена регистрации центральный сайт в центре администрирования SharePoint
  
1. В центре администрирования SharePoint в разделе **сайтов**щелкните **сайты Active**. 
    
2. Выберите сайт, щелкните **центральный сайт**и нажмите кнопку **Отменить регистрацию как центральный сайт**. 
    
3. Нажмите кнопку **ОК**. 
    
[Дополнительные сведения о сайтах сервера-концентратора](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
