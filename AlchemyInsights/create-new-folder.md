# <a name="create-a-new-folder"></a>Создание папки

1. В разделе **папки** в области навигации выберите пункт **создать папку**.

2. Введите имя папки и нажмите клавишу ВВОД.

Чтобы узнать больше, ознакомьтесь [со статьей Создание папки в Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
