# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>Использование Microsoft Edge для удаленной отладки устройства с Android с компьютера разработчика

Для удаленной отладки динамического содержимого на устройстве с компьютера Android выполните следующие действия.

1. Настройка устройства с Android для удаленной отладки и ее обнаружение на компьютере разработчика.
2. На компьютере разработчика проверьте и отлаживать Live Contents на устройстве с Android.
3. Содержимое ролика Live из устройства Android на экземпляр Девтулс на компьютере разработчика.
