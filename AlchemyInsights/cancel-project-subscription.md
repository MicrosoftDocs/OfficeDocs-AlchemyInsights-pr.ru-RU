# <a name="cancel-project-subscription"></a>Отмена подписки на проект

Администраторы могут отменить подписку на проект для своей организации с помощью **центра администрирования Microsoft 365**. 

1. Выберите параметр **выставление счетов** .

2. Выберите [продукты & службы](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/subscriptions) , чтобы просмотреть все подписки Организации.

3. Откройте план проекта, который требуется отменить.

4. Нажмите кнопку **Отменить подписку**.
