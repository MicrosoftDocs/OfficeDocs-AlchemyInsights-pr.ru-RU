# <a name="deploy-ad-fs"></a>Развертывание AD FS

Развертывание служб федерации Active Directory (AD FS) использует вашу локальной инфраструктуру для проверки подлинности пользователей для служб Office 365. Благодаря федеративному входу вы можете позволить пользователям вошел в службы Office 365 и приложения SAAS, интегрированные с Azure Active Directory (Azure AD). Федераированный вход позволяет проверить подлинность пользователей в локальной службе Active Directory с помощью AD FS. Кроме того, в корпоративной сети пользователям не потребуется повторно вводить свои пароли.

Помощник по развертыванию [AD FS](https://go.microsoft.com/fwlink/?linkid=2071178) предоставляет пошаговые инструкции по развертыванию локальной инфраструктуры AD FS, которая обеспечивает проверку подлинности пользователей для служб Microsoft 365 и Office 365. С помощью этого руководства ваша организация может просмотреть компоненты и требования AD FS, приобрести и установить SSL-сертификаты, необходимые для развертывания, и установить необходимый прокси-сервер веб-приложения.
