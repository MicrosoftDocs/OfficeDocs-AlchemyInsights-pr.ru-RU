# <a name="microsoft-teams-exploratory-experience"></a>Исследовательский интерфейс Microsoft Teams

Исследовательский интерфейс Microsoft Teams позволяет пользователям в вашей организации, у которых есть служба Azure Active Directory (AAD) и не лицензироваться с помощью Teams, выполнять произвольную работу Teams. Администраторы могут включить или отключить эту функцию для пользователей в своей организации. (Более ранняя коммерческая облачная пробная версия Майкрософт называется произвольным опытом работы Teams.)

Дополнительную информацию можно узнать [в статье управление исследовательской лицензией Microsoft Teams](https://docs.microsoft.com/microsoftteams/teams-exploratory/).
