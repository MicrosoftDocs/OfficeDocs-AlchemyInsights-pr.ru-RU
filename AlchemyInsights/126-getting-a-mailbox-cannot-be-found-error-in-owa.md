# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a><span data-ttu-id="9ffb5-102">Ошибка при возврате почтового ящика "не найдено" в Outlook в Интернете</span><span class="sxs-lookup"><span data-stu-id="9ffb5-102">Getting a mailbox not found error in Outlook on the web?</span></span>

<span data-ttu-id="9ffb5-p101">Если вы используете Outlook в Интернете и **не можете найти почтовый ящик по** ошибке, учетная запись, используемая для подключения к Outlook в Интернете, не имеет лицензии на Exchange Online, поэтому с этой учетной записью не связан ни один почтовый ящик. Администратор может назначить лицензию вашей учетной записи, выполнив указанные ниже действия.</span><span class="sxs-lookup"><span data-stu-id="9ffb5-p101">If you're using Outlook on the web and you get a **Mailbox couldn't be found for** error, the account that you used to connect to Outlook on the web doesn't have an Exchange Online license and therefore, no mailbox is associated with the account. Your admin can assign a license to your account by following these steps:</span></span> 
  
1. <span data-ttu-id="9ffb5-105">Откройте [центр администрирования Microsoft 365](https://portal.office.com/adminportal/home#/homepage) и в разделе **Активные пользователи**выберите **изменить пользователя**.</span><span class="sxs-lookup"><span data-stu-id="9ffb5-105">Open the [Microsoft 365 admin center](https://portal.office.com/adminportal/home#/homepage) and under **Active users**, select **Edit a user**.</span></span>
    
2. <span data-ttu-id="9ffb5-p102">На открывшейся странице **изменение пользователя** выберите пользователя. На открывшейся странице свойств пользователя щелкните **изменить** для пункта **лицензии на продукты**.</span><span class="sxs-lookup"><span data-stu-id="9ffb5-p102">In the **Edit a user** page that opens, select the user. In the user properties page that opens, click **Edit** for **Product licenses**.</span></span>
    
3. <span data-ttu-id="9ffb5-p103">На открывшейся странице **лицензии на продукты** выберите нужное значение **расположения** и назначьте лицензию, которая содержит Exchange Online (разверните лицензию, чтобы просмотреть сведения о ней). Когда все будет готово, нажмите кнопку **сохранить**.</span><span class="sxs-lookup"><span data-stu-id="9ffb5-p103">In the **Product licenses** page that opens, select the appropriate **Location** value, and assign a license that contains Exchange Online (expand the license to see its details). When you're finished, click **Save**.</span></span>
