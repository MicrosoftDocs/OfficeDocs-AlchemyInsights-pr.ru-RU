# <a name="in-app-support-in-outlook-for-mac"></a>Поддержка встроенных приложений в Outlook для Mac

Чтобы получить поддержку по проблемам с приложением для Outlook для Mac, откройте Outlook и щелкните ссылку \> "Справка". Введите вопрос, а участник группы поддержки Outlook для Mac будет общаться с вами. [Обращение в службу поддержки Outlook для Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517)
