# <a name="change-your-profile-name"></a>Изменение имени профиля

1. Перейдите на [свой Microsoft сведения учетной записи](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Выберите команду **изменить имя**. 
    
3. Введите имя и фамилию и выберите команду **Сохранить**. 
    
Подождите до 24 часов для изменения имени профиля вступили в силу.
  
Дополнительные сведения содержатся в разделе [Изменение фотографий профиля или имя в Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
