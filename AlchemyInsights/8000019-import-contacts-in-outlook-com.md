# <a name="import-contacts"></a>Импорт контактов

Прежде чем импортировать контакты для Outlook.com, вам потребуется экспортировать их от других электронной почты поставщика как CSV-файл. Следуйте приведенным ниже инструкциям.
  
1. И в нижней части страницы выберите **пользователей**. 
    
2. В строке меню выберите пункт **Управление** \> **Импорт контактов**. 
    
3. Выберите **Gmail** (даже если CSV-файл не могла возникнуть из Gmail), так как тип файла .csv используется Gmail. 
    
4. Выберите **Обзор**, перейдите к папке, в которой был сохранен экспортированный контактов и выберите команду **Открыть**.
