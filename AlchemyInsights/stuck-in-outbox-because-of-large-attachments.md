# <a name="fix-messages-that-are-stuck-in-the-outbox-with-sara"></a>Устранение сообщений, задержанных в папке "Исходящие" с SaRA

Для начала рекомендуется выполнить сценарий ["проблемы с отправкой, получением или поиском сообщений электронной почты"](https://aka.ms/SaRA-OutlookSendReceive) на компьютере, на котором работает программа " [Помощник по поддержке и восстановлению Майкрософт (SaRa)](https://diagnostics.office.com/#/) ".
