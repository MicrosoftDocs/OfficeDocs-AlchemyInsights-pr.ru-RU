# <a name="microsoft-search-setup-guide"></a>Руководство по настройке microsoft Search

Чтобы настроить и настроить Microsoft [Search,](https://go.microsoft.com/fwlink/?linkid=2153798) используйте руководство по настройке Microsoft Search. Хотите ли вы пилотное решение для группы пользователей или выкатите его всем в вашей организации. Вы назначите администраторам поиска и редакторам поиска, а затем настроите интерфейс поиска для пользователей с помощью ответов и дополнительных параметров, таких как добавление расширения Bing в Chrome или настройка Bing в качестве поисковой системы по умолчанию.
