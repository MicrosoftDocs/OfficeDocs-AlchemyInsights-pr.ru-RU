# <a name="copy-and-paste-in-outlookcom"></a>Копирование и вставка в Outlook.com

В Outlook.com нет кнопок "Копировать" и "вставить", но можно использовать сочетания клавиш:

**На компьютере:** CTRL + C для копирования, CTRL + X для вырезания и CTRL + V для вставки.

**На компьютере Mac:** CMD + C для копирования, CMD + X для вырезания, а CMD + V — для вставки.
