# <a name="delete-a-sharepoint-site-collection"></a>Удаление семейства сайтов SharePoint

1. В классический центра администрирования SharePoint на странице семейств сайтов установите флажок рядом с элементом коллекции веб-сайтов.
    
2. На вкладке семейств веб-сайтов выберите команду **Удалить**.
