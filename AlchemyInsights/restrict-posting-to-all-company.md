# <a name="restrict-posting-to-all-company"></a>Ограничение размещения для всех компаний

Можно ограничить все [публикации компании](https://support.microsoft.com/office/restrict-all-company-posts-in-yammer-3219d2ae-db15-4c9f-9dd2-28559ae39a97) в Yammer с помощью параметров, доступных администраторам. Пользователи по-прежнему смогут отвечать на потоки, запущенные администраторами, если администраторы явно не закроют [беседы.](https://support.microsoft.com/office/pin-close-and-report-conversations-in-yammer-62a5fbc2-ff1b-4418-9334-d2b4b17062cb) В настоящее время невозможно ограничить размещение в других сообществах Yammer.
