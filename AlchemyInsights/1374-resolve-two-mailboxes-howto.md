# <a name="a-user-has-two-mailboxes"></a>У пользователя есть два почтовых ящиков

Гибридные среды, использующие Azure Active Directory подключение (AAD подключение) или DirSync случайно могут привести к пользователю иметь два почтовых ящиков: одна в локальной организации, другая — в облаке.
  
Чтобы устранить эту проблему, видеть [Без возможности восстановления очистить предыдущие сведения почтового ящика](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
