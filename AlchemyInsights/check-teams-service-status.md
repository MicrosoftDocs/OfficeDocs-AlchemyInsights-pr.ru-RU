# <a name="check-teams-service-status"></a>Проверка состояния службы Teams

Чтобы проверить текущее состояние службы Microsoft Teams, перейдите на панель мониторинга работоспособности в Центре администрирования Microsoft 365.

Дополнительные сведения см. в статье [Проверка работоспособности службы Microsoft 365](https://docs.microsoft.com/office365/enterprise/view-service-health).
