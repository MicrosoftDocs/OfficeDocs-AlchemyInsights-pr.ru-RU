# <a name="prepaid-renewal"></a>Будущих обновления

1. В Центре администрирования откройте страницу [Подписки](https://go.microsoft.com/fwlink/p/?linkid=842054) или выберите **Выставление счетов** \> **Подписки**.
    
2. Выберите будущих подписку, для которой нужно выполнить расширение, а затем выберите команду **расширить**.
    
3. Выберите параметр для **ключа продукта**. (В случае ключ продукта, приобрести в розничном месте может имеется возможность расширить другие методы оплаты.)
    
4. Введите ключ продукта.
    
5. Убедитесь, что указаны правильные данные, а затем нажмите кнопку **Далее**.
    
6. Выберите переключатель, чтобы увеличить дату окончания существующих лицензий и нажмите кнопку **Далее**.
