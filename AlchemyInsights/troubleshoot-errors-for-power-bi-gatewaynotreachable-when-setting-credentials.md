# <a name="gatewaynotreachable-when-setting-credentials"></a>Гатевайнотреачабле при задании учетных данных

При попытке задать учетные данные для источника данных можно столкнуться с Гатевайнотреачабле. Это может быть вызвано устаревшим шлюзом. Установите последний шлюз и повторите попытку.

Дополнительные сведения:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
