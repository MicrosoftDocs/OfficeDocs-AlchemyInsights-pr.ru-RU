# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Приступая к ошибке разрешений при открытии общего почтового ящика в Outlook в Интернете?

Если возникнет ошибка **что-то случилось. У вас нет разрешения на открытие этого почтового ящика.** При попытке открыть общий почтовый ящик в Outlook в Интернете, учетной записи нет разрешения на доступ к общему почтовому ящику. Администратор организации можно назначить разрешения учетной записи общего почтового ящика, следуя инструкциям, приведенным в разделе [Создание общего почтового ящика](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).
