# <a name="unable-to-attach-or-upload-files-from-facebook"></a>Не удается присоединить или отправить файлы из Facebook

В связи с изменением условий и условий Facebook, начиная с 23 октября 2019, вы больше не сможете загружать вложения прямо из Facebook, и Facebook больше не будет отображаться в списке облачных расположений. 

Вы можете управлять списком других облачных расположений в [учетных записях хранения](https://go.microsoft.com/fwlink/?linkid=2111075).

Дополнительные сведения об [учетных записях хранения](https://support.office.com/article/477cb7cc-5732-4c40-8f23-30472de8138a).
