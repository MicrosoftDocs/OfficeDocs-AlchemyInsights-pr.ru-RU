# <a name="setting-up-visio"></a>Настройка Visio

Похоже, вам нужна помощь в настройке Visio с подпиской на Microsoft 365.
  
Примечание для пользователей Mac: продукт Visio в настоящее время недоступен для MacOS. Если у вас есть компьютер, вы можете установить программное обеспечение, выполнив указанные ниже действия.
  
Вы можете выполнить действия, описанные в [этой статье](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) , чтобы завершить настройку.
