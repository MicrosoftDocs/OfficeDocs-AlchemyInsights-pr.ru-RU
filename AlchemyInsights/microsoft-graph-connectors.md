# <a name="microsoft-graph-connectors"></a>Соединители Microsoft Graph

С помощью Microsoft Graph Connectors в Организации можно индексировать сторонние данные, которые будут отображаться в результатах поиска Майкрософт.

- [Соединители корпорации Майкрософт](https://docs.microsoft.com/microsoftsearch/connectors-gallery#Microsoft)
- [Соединители наших партнеров](https://docs.microsoft.com/microsoftsearch/connectors-gallery#Partners)

Более подробную информацию можно узнать в разделе [Обзор соединителей Microsoft Graph](https://docs.microsoft.com/microsoftsearch/connectors-overview).
