# <a name="duplicate-object-synchronization-errors"></a>Дублирующиеся ошибки синхронизации объектов

Если вы получаете следующее ошибки во время синхронизации:
  
- Синхронизированных объектов с тем же адресом прокси-сервер уже существует в каталоге Microsoft Online Services.
    
- Аналогично сообщение об UserPrincipalName.
    
Используйте новый работоспособности AADConnect диагностики для устранения ошибок повторяющихся объектов. Дополнительные сведения можно [Диагностика и устранение ошибок синхронизации дублирования атрибута](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
