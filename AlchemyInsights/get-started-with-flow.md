# <a name="get-started-with-microsoft-flow"></a>Начало работы с Microsoft Flow

С помощью Microsoft Flow можно выполнять следующие действия:
  
- Простой поиск шаблонов и служб.
    
- Выберите одну из наиболее популярных служб.
    
- Ознакомьтесь с обзором каждого из этих поступлений.
