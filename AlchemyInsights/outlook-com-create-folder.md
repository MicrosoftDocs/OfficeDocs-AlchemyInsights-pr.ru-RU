# <a name="create-a-folder"></a>Создание папки

1. В нижней части области папок выберите пункт **создать папку**. Возможно, вам потребуется прокрутить панель папок вниз, чтобы увидеть кнопку **создать папку** .
2. В текстовом поле введите имя папки и нажмите клавишу ВВОД.

Чтобы узнать больше, ознакомьтесь [со статьей Создание папки в Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
