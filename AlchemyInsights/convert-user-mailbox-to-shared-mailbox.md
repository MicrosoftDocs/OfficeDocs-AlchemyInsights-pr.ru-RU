# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Преобразования почтового ящика пользователя в почтовый ящик помещения или оборудования

Также можно преобразовать почтового ящика от типа один почтовый ящик в другую, но вам придется для выполнения этих задач с помощью Exchange Online PowerShell. Извлечь [Преобразование почтового ящика](https://go.microsoft.com/fwlink/p/?LinkId=832875) для получения дополнительных сведений.
