# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Преобразования почтового ящика пользователя в почтовый ящик помещения или оборудования

Вы также можете преобразовать почтовый ящик из одного типа почтового ящика в другой, но вам потребуется выполнить эти задачи с помощью Exchange Online PowerShell. Для получения дополнительных сведений ознакомьтесь [с](https://go.microsoft.com/fwlink/p/?LinkId=832875) разОбразованием почтового ящика.
