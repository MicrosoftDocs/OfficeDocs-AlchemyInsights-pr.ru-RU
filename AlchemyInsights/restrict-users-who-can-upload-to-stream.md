# <a name="restrict-users-who-can-upload-to-stream"></a>Ограничение числа пользователей, которые могут отправляться в поток

В качестве организации вы можете ограничить сотрудников, которые могут отправлять видеоролики. По умолчанию все пользователи в организации могут отправлять видеоролики.

- [Ограничение сотрудников, которые могут создавать содержимое в Microsoft Stream](https://docs.microsoft.com/stream/restrict-uploaders)
