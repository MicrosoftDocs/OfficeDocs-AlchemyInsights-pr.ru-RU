# <a name="problems-sharing-with-outlook-customer-manager"></a>Проблемы, связанные с обменом данными с клиентом Outlook Manager

Если у вас возникли проблемы с клиентом Outlook Manager, рекомендуем сначала очистить кэш браузера. Например, в Internet Explorer вы бы:

1. Перейдите **к общим**> параметрам в кэшах истории просмотра и  >     >  **базах данных.**
2. Удаление файлов кэша для веб-сайта "outlookapps.com".
3. Перезапустите Outlook.

Дополнительные сведения о совместном использовании см. [в разделе Share information in Outlook Customer Manager.](https://techcommunity.microsoft.com/t5/outlook-blog/sharing-how-to-keep-your-colleagues-in-the-loop/ba-p/35710)
