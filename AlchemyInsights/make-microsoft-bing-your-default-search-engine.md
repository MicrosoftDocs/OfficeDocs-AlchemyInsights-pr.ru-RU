# <a name="make-microsoft-bing-your-default-search-engine"></a>Сделайте Microsoft Bing поисковой машиной по умолчанию

Чтобы сделать Bing вашей поисковой системы по умолчанию для Microsoft Edge, Google Chrome или Internet Explorer, следуйте инструкциям в [Сделать Bing поисковой системы](https://go.microsoft.com/fwlink/?linkid=2148834)по умолчанию .
