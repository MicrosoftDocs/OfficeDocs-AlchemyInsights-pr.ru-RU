# <a name="changes-to-external-user-access"></a>Изменения для доступа внешних пользователей

Запуск 23 марта 2018, внешних пользователи больше не смогут для доступа к контенту, общего все, всем пользователям, прошедшим проверку подлинности или всех пользователей формы, даже если они приглашены или добавить в авторизованный группы. Внешние пользователи смогут только для доступа к контенту, общий с пользователя или группы, к которым принадлежит пользователь. Если вы хотите сохранить существующую, вы можете найти:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
