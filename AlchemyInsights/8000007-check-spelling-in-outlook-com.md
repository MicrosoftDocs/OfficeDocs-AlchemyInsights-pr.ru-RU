# <a name="use-your-browser-to-check-spelling"></a>Использование браузера для проверки орфографии

Нет в Outlook.com отсутствует функция проверки правописания. Для проверки правописания, необходимо использовать веб-браузера. Проверка правописания доступна в пограничного сервера Microsoft Internet Explorer 10 и более поздних версий и текущие версии Firefox, Chrome и Safari. Проверьте параметры веб-обозревателя для получения дополнительных сведений о проверке правописания.
