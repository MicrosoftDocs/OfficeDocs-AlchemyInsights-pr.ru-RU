# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>В Outlook для Mac возникают повторяющиеся приглашения на пароль?

Вы можете предоставить более специализированную справку по Outlook для Mac непосредственно в приложении.
  
Перейдите к разделу Mac затронутого пользователя, откройте Outlook и щелкните ссылку **Справка** \> **Поддержка**.
  
Дополнительные сведения об использовании Outlook для Mac в службе поддержки клиентов в приложениях можно найти в этой статье: [Новая функция поддержки контактов в outlook 2016 для Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
