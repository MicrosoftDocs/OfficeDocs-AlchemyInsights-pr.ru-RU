# <a name="add-a-guest-to-microsoft-teams"></a>Добавление гостя в Microsoft Teams

Иногда вам нужно сотрудничать с людьми, которые не являются в организации вашей команды. Часто вы будете делать это по электронной почте, но если вы хотите сотрудничать с ними в Teams, вы можете добавить их в качестве гостей.

Дополнительные сведения см. в [добавлении гостей в команду Teams.](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop)

**Примечание:** Вы должны быть владельцем команды, чтобы добавить гостя в Teams (также см. [в руб. Set guest permissions).](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)
