# <a name="setting-up-a-hybrid-environment"></a>Настройка гибридной среды

Чтобы пользователи работали без проблем независимо от того, хранятся ли их данные на локальных серверах или в Office 365, может потребоваться настроить гибридную среду. Чтобы получить общие сведения о возможностях, изучите [интеграцию Office 365 с локальными средами](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
