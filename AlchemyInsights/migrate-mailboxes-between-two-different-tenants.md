# <a name="migrate-mailboxes-between-two-different-tenants"></a>Миграция почтовых ящиков между двумя разными клиентами

Вам может потребоваться перенести почтовые ящики между двумя разными клиентами, например при объединении Организации. Эти миграции должны быть тщательно запланированы. Ознакомьтесь со следующими рекомендациями:
  
[Перенос почтовых ящиков из одного клиента в другой](https://docs.microsoft.com/Exchange/mailbox-migration/migrate-mailboxes-across-tenants)
