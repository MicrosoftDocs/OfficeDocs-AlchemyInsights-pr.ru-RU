# <a name="site-and-content-structure-removed"></a>Структура сайта и контента удалена

Начиная с 2018 октября, страница контент и структура сайта больше не будет поддерживаться или получать обновления компонентов. Ссылка параметры на страницу Ситеманажер. aspx будет удалена из SharePoint Online, а прямой доступ будет ограничен администраторами семейства веб-сайтов до 2019 марта, после чего страница контент и структура сайта будет полностью отменена. 

Примечание. в качестве обходного пути до полного отказа компонента Администраторы семейства веб-сайтов могут перейти на портал Office 365-_Гт_ нажмите кнопку SharePoint плитка — _Гт_ добавьте или измените URL-адрес, заменив "/шарепоинт.аспкс" на "/Ситеманажер.аспкс". 


Используйте перемещение и копирование файлов для управления контентом и структурой сайта. Дополнительные сведения см. в статье [копирование файлов и папок между OneDrive и сайтАми SharePoint](https://support.office.com/en-us/article/copy-files-and-folders-between-onedrive-and-sharepoint-sites-67a6323e-7fd4-4254-99a8-35613492a82f). 

Для получения дополнительных сведений обратитесь к разделу [Работа с контентом и структурой сайта](https://support.office.com/en-us/article/Work-with-site-content-and-structure-30fcaad9-02b1-4347-8b03-e1ccc5a4c19f).
