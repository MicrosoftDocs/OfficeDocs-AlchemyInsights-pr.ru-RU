# <a name="we-cant-get-your-files"></a>Мы не можем получить файлы

Пользователи могут видеть эту ошибку, если они не являются членами группы. Добавьте пользователя в группу, чтобы предоставить ему доступ к групповому сайту.

- Дополнительные сведения о добавлении пользователя в команду см. в дополнительных сведениях о добавлении пользователей [в команду Teams.](https://support.office.com/article/add-people-to-a-team-aff2249d-b456-4bc3-81e7-52327b6b38e9)
- Дополнительные сведения о непосредственном совместном использовании сайта см. в разделе [Share a site.](https://support.office.com/article/Share-a-site-958771A8-D041-4EB8-B51C-AFEA2EAE3658)

Эта ошибка также может возникнуть, если группа не подключена к команде SharePoint. Дополнительные сведения о подключении группы к сайту SharePoint см. в раздел [Подключение к группе Microsoft 365](https://docs.microsoft.com/sharepoint/dev/transform/modernize-connect-to-office365-group)
