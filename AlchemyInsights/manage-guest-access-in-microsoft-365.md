# <a name="manage-guest-access-in-microsoft-365"></a>Управление гостевым доступом в Microsoft 365

Полные инструкции по настройке общего доступа для гостей в Группах, SharePoint и Teams см. в следующих статьях: 

- [Совместная работа с гостями в команде](https://docs.microsoft.com/microsoft-365/solutions/collaborate-as-team?view=o365-worldwide) с помощью **Microsoft Teams**. 

- [Совместная работа с гостями на сайте](https://docs.microsoft.com/microsoft-365/solutions/collaborate-in-site?view=o365-worldwide) с помощью **SharePoint**. 

- [Совместная работа с гостями во внешних группах](https://docs.microsoft.com/yammer/work-with-external-users/create-and-manage-external-groups?redirectSourcePath=%252farticle%252f9ccd15ce-0efc-4dc1-81bc-4a424ab6f92a.aspx) с помощью **Yammer**. 

- [Управление гостевым доступом](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-guest-access-in-groups?view=o365-worldwide) в **группе**.
