# <a name="change-the-font-size-in-a-message"></a>Изменение размера шрифта в сообщении

1. Если вы уже ввели сообщение, выберите текст.
2. Выберите значок  **формата** под сообщением (первый значок).
3. Выберите значок  **размера шрифта**  над сообщением (это значок седьмого).
4. Выберите размер.
