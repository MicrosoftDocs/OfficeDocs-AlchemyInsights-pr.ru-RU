# <a name="create-custom-content-types"></a>Создание настраиваемых типов контента

1. Выберите нужный список на панели быстрого запуска или щелкните **Параметры**  (значок шестеренки), а затем щелкните  **Параметры сайта**.
2. В **коллекции веб-дизайнера**щелкните типы **контента сайта** .
3. На странице "типы контента сайта" нажмите кнопку Создать.
4. На странице новый тип контента сайта введите сведения о настраиваемом типе контента.

Более подробную информацию можно узнать в разделе  [Настраиваемые типы контента](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
