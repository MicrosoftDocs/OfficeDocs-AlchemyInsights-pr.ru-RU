# <a name="create-a-folder"></a>Создание папки

1. В левой области выберите **создать папку** рядом с папкой. **** 
    
2. В поле Новая папка введите имя папки и нажмите клавишу ВВОД.
    
Чтобы узнать больше, ознакомьтесь [со статьЕй создание папки в Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
