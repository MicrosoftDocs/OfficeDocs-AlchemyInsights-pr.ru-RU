# <a name="download-files-and-folders-from-onedrive-or-sharepoint"></a>Скачивание файлов и папок из OneDrive или SharePoint

Вы можете скачивать файлы и папки из Microsoft OneDrive, SharePoint в Microsoft 365 или SharePoint Server 2019 на свой компьютер всего несколькими щелчками мыши. Дополнительные сведения см. в статье [Скачивание файлов и папок из OneDrive или SharePoint](https://support.microsoft.com/office/download-files-and-folders-from-onedrive-or-sharepoint-5c7397b7-19c7-4893-84fe-d02e8fa5df05).
