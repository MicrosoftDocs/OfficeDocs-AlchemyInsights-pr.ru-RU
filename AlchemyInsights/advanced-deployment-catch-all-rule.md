# <a name="deploy-office-apps"></a>Развертывание приложений Office

Для развертывания приложений Office для нескольких конечных пользователей с помощью средства развертывания Office, System Center Configuration Manager, в составе образа диска операционной системы или сценария на общем компьютере с использованием Windows Server со службами удаленных рабочих столов (RDS) или терминалом Службы, используйте пошаговое руководство по **развертыванию Office 365 профессиональный плюс**.
  
Чтобы приступить к работе, убедитесь, что вы выполнили вход в качестве администратора клиента или глобального администратора, а затем щелкните [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance)здесь:.
