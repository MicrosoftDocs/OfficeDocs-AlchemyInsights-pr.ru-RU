# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a><span data-ttu-id="37812-102">Ошибка при возврате почтового ящика "не найдено" в Outlook в Интернете</span><span class="sxs-lookup"><span data-stu-id="37812-102">Getting a mailbox not found error in Outlook on the web?</span></span>

<span data-ttu-id="37812-103">Если вы используете Outlook в Интернете и **не можете найти почтовый ящик по** ошибке, учетная запись, используемая для подключения к Outlook в Интернете, не имеет лицензии на Exchange Online, поэтому с этой учетной записью не связан ни один почтовый ящик.</span><span class="sxs-lookup"><span data-stu-id="37812-103">If you're using Outlook on the web and you get a **Mailbox couldn't be found for** error, the account that you used to connect to Outlook on the web doesn't have an Exchange Online license and therefore, no mailbox is associated with the account.</span></span> <span data-ttu-id="37812-104">Администратор может назначить лицензию вашей учетной записи, выполнив указанные ниже действия.</span><span class="sxs-lookup"><span data-stu-id="37812-104">Your admin can assign a license to your account by following these steps:</span></span>

1. <span data-ttu-id="37812-105">Откройте [центр администрирования Microsoft 365](https://portal.office.com/adminportal/home#/homepage) и перейдите к разделу **Активные пользователи** в разделе **Пользователи** и выберите пользователя, который видит ошибку.</span><span class="sxs-lookup"><span data-stu-id="37812-105">Open the [Microsoft 365 admin center](https://portal.office.com/adminportal/home#/homepage) and go to **Active users** under the **Users** section, and select the user who is seeing the error.</span></span>

2. <span data-ttu-id="37812-106">На открывшейся странице пользователя перейдите к разделу **лицензии и приложения** , выберите нужное значение **расположения** и назначьте лицензию, содержащую Exchange Online (разверните лицензию, чтобы просмотреть сведения о ней).</span><span class="sxs-lookup"><span data-stu-id="37812-106">In the user page that opens, go to the **Licenses and Apps** section, select the appropriate **Location** value, and assign a license that contains Exchange Online (expand the license to see its details).</span></span> <span data-ttu-id="37812-107">По завершении нажмите кнопку **сохранить изменения**.</span><span class="sxs-lookup"><span data-stu-id="37812-107">When you're finished, click **Save changes**.</span></span>
