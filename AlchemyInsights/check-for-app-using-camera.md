# <a name="check-for-app-using-camera"></a>Проверка приложения с помощью камеры

1. Нажмите кнопку **Пуск**, выберите **параметры > конфиденциальность >ная Камера** (MS-параметры: Конфиденциальность-веб-камера)
2. В списке Application (приложение) найдите приложение с параметром "используется в данный момент" и закройте приложение.
