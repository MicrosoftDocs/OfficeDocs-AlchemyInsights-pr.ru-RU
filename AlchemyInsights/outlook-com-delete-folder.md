# <a name="delete-a-folder-in-outlookcom"></a>Удаление папки в Outlook.com

1. В области слева щелкните правой кнопкой мыши папку или подпапку, которую нужно удалить, и выберите команду **Удалить папку**.
2. Нажмите кнопку **ОК** , чтобы подтвердить.

Дополнительные сведения см. [в статье работа с папками сообщений в Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
