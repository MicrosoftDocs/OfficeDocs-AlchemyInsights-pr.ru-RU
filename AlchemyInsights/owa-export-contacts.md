# <a name="export-contacts"></a>Экспорт контактов

1. В нижней части страницы выберите значок люди, чтобы перейти на страницу люди.
2. На панели инструментов выберите **Управление** > **экспортом контактов**. 
3. Выберите, чтобы экспортировать все контакты или только контакты из определенной папки, а затем нажмите кнопку **Экспорт**.
4. В нижней части страницы нажмите кнопку **сохранить** , чтобы сохранить файл Contacts. csv в папке Downloads по умолчанию. Или нажмите стрелку рядом с пунктом **сохранить** и выберите **Сохранить как**.

Чтобы узнать больше, ознакомьтесь [со статьей Экспорт контактов из Outlook](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web).
