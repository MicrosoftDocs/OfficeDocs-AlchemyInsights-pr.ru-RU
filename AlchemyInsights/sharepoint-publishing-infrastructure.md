# <a name="sharepoint-publishing-infrastructure"></a>Инфраструктура публикации SharePoint


<p>SharePoint включает классический набор функций, позволяющих создавать и публиковать расширенные веб-страницы для интрасети. Активируйте инфраструктуру публикации SharePoint на уровне семейства веб-сайтов, чтобы включить их.</p>  <p>Для получения дополнительных сведений перейдите по приведенным ниже ссылкам.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Включение публикации</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Функции публикации</a></li>  </ul>  <p>Примечания.</p>  <ul>  <li>Шаблоны классических сайтов публикации по умолчанию включают функции публикации.</li>  <li>Классические функции публикации недоступны на современных сайтах. Современные сайты предоставляют более новую альтернативу по умолчанию.</li>  </ul>
