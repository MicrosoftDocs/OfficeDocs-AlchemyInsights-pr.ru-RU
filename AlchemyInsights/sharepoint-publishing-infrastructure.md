# <a name="sharepoint-publishing-infrastructure"></a>Инфраструктура публикации SharePoint


SharePoint включает классический набор функций, позволяющих создавать и публиковать расширенные веб-страницы для интрасети. Активируйте инфраструктуру публикации SharePoint на уровне семейства веб-сайтов, чтобы включить их.

Для получения дополнительных сведений перейдите по приведенным ниже ссылкам.

[Включение публикации](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Функции публикации](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Примечания.

- Шаблоны классических сайтов публикации по умолчанию включают функции публикации.

- Классические функции публикации недоступны на современных сайтах. Современные сайты предоставляют более новую альтернативу по умолчанию.
