# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Узнайте о сопоставлении между политиками Google Chrome и политиками Microsoft Edge

Политики Google Chrome со картой с соответствующими политиками Microsoft Edge, поддерживаемые в версии 80 браузера. Чтобы узнать больше, [см. сопоставление политик Google Chrome и Microsoft Edge.](https://go.microsoft.com/fwlink/?linkid=2141933)
