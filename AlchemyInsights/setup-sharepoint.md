# <a name="set-up-sharepoint"></a>Настройка SharePoint

[Руководство по настройке SharePoint](https://go.microsoft.com/fwlink/?linkid=2071425) поможет вам настроить хранилище документов SharePoint и управление контентом, создавать сайты, настраивать внешний общий доступ, переносить данные и настраивать дополнительные параметры, а также управлять участием пользователей и общением в Организации.

Выполните действия по настройке политик разрешений для совместного использования содержимого, выберите средства синхронизации миграции, а также включите параметры безопасности для среды SharePoint.
