# <a name="rename-a-site"></a>Переименование сайта

Чтобы изменить имя сайта (например, сменить https://contoso.sharepoint.com/sites/sample2 на [](https://go.microsoft.com/fwlink/?Linkid=2018691)), нужно создать новый сайт, а затем перенести в него весь контент старого сайта. Сведения о перемещении элементов в библиотеках документов см. в статье Перемещение или копирование элементов из библиотеки документов SharePoint.
  
Примечание: Если вы пытаетесь изменить имя домена (то есть, contoso часть URL-адреса выше) в SharePoint Online URL-адрес, это невозможно. Просмотрите, [изменять домены](https://go.microsoft.com/fwlink/?Linkid=2018696).
