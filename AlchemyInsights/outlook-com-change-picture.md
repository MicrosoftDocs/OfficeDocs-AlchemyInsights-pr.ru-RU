# <a name="change-your-microsoft-account-picture"></a>Изменение аватара учетной записи Майкрософт

1. Перейдите к [сведениям учетной записи Майкрософт](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Выберите команду **Изменить рисунок**.
3. Выберите **Создать рисунок**.
4. Выберите изображение, которое нужно использовать, и нажмите кнопку **Открыть**.
5. Перетащите поле изображения или измените его размер, а затем нажмите кнопку **сохранить**.

Чтобы изменения фотографии профиля вступили в последствия, оставьте до 24 часов.
