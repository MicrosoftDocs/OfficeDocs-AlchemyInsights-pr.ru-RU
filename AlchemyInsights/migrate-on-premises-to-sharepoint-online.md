# <a name="migrate-on-premises-to-sharepoint-online"></a>Миграция локальной сети в SharePoint Online

Для миграции из локальной фермы SharePoint в SharePoint Online можно использовать средство миграции SharePoint. Дополнительные сведения о средстве можно найти [в статье миграция в SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)
