# <a name="enable-versioning-for-a-list-or-library"></a>Включение управления версиями для списка или библиотеки

1. Перейдите к списку или библиотеке.
    
2. Нажмите кнопку **Параметры**и нажмите кнопку **Параметры библиотеки** или **списка параметров**.
    
3. Щелкните **Параметры управления версиями**.
    
4. Выберите нужные параметры и нажмите кнопку **ОК**.
