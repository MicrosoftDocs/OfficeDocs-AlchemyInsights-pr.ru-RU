# <a name="your-users-mailbox-is-being-created"></a>Идет создание почтового ящика пользователя

Для того чтобы новый почтовый ящик был готов к использованию, он может занять до часа.
  
Если почтовый ящик пользователя подготовлен к работе в течение более часа, можно попробовать удалить, а затем повторно создать пользователя, или использовать параметр "придать нам Звонок", чтобы поговорить с инженером службы поддержки.
