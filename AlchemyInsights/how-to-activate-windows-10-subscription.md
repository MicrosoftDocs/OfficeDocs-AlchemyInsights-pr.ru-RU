# <a name="activating-windows-10-subscriptions"></a>Активация подписок на Windows 10

Microsoft 365 бизнес премиум, Office 365 для образования, Microsoft 365 и Enterprise предоставляют цифровые лицензии для Windows 10 в рамках подписки. Чтобы активировать подписку, на вашем устройстве уже должна быть активирована Windows 10 Pro (1703 или более поздней версии). Полный список требований по активации подписки на Windows 10 можно найти в статье [требования к активации подписки на Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
