# <a name="where-is-power-pivot"></a>Где находится Power Pivot?

Существует три причины, почему вы не видите Power Pivot.
  
 **Нет вашей версии Excel**
  
Не все версии Excel содержат Power Pivot. Необходимо выполнить обновление до версии Office, которая включает в себя Power Pivot. [Контрольный список поддерживаемых версий.](https://support.office.com/article/aa64e217-4b6e-410b-8337-20b87e1c2a4b.aspx)
  
 **Еще не включено Power Pivot**
  
Если вы работаете в версии Excel, которая поддерживает Power Pivot, проверьте, что надстройка включено:
  
1. Выберите **файл** \> **Параметры** \> **Надстроек**.
    
2. В **Управление** раскрывающемся списке пункт **надстройки COM** и нажмите кнопку **Перейти**.
    
3. В диалоговом окне **Надстройки Com** установите флажок рядом с элементом **Microsoft PowerPivot для Excel**и нажмите кнопку **ОК**. 
    
 **У вас есть несколько подписок или учетных записей**
  
Можно создать различные подписки Office 365 назначены разные учетные записи. Убедитесь, что вход выполнен для учетной записи с версии Excel, которая поддерживает Power Pivot. Проверьте список [поддерживаемых версий](https://support.office.com/article/aa64e217-4b6e-410b-8337-20b87e1c2a4b.aspx)и следуйте инструкциям, чтобы [переключиться между разные учетные записи Office](https://support.office.com/article/b9582171-fd1f-4284-9846-bdd72bb28426.aspx#BKMK_WebSwitchAccounts).
