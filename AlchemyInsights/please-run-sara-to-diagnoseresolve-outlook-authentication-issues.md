# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Использование помощника SaRA для диагностики и устранения проблем с проверкой подлинности в Outlook

**Примечание**: Проверьте, включены ли для вашей организации [Параметры безопасности по умолчанию](https://aka.ms/securitydefaults) . Если ваш клиент создан после 21 октября 2019, а Outlook будет периодически запрашивать пароль, в клиенте могут быть включены **Параметры безопасности по умолчанию** .

Мы настоятельно рекомендуем использовать [Outlook для](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) устранения неполадок диагностики паролей на затронутом компьютере, чтобы устранить проблемы, в результате которых Outlook постоянно запрашивает пароль. Эта диагностика [SaRA](https://diagnostics.office.com/#/) выполняет автоматические проверки и возвращает возможные решения, которые можно использовать для устранения обнаруженных проблем.
