# <a name="in-app-support-in-outlook-for-windows"></a>Поддержка встроенных приложений в Outlook для Windows

Администратор клиента может получить поддержку в приложении Outlook для Windows. В Outlook выберите пункт **Справка**  >  **Поддержка**. Введите вопрос, и участник группы поддержки Outlook для Windows ответит на Ваше сообщение.
