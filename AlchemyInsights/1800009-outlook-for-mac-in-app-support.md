# <a name="in-app-support-in-outlook-for-mac"></a>Поддержка из приложения в Outlook для Mac

Для получения поддержки в приложение Outlook для Mac проблемы, откройте программу Outlook и нажмите кнопку Справка, \> обратитесь в службу поддержки. Введите вопрос и должна быть членом Outlook для Mac, группа поддержки будет вами связаться. [Обратиться в службу поддержки в Outlook для Mac](https://go.microsoft.com/fwlink/?linkid=2002400&amp;clcid=0x409)
