# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Получение учетной записи этот параметр отключен ошибка в Outlook в Интернете?

Если возникнет ошибка **отключил свою учетную запись** с `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` при попытке открыть Outlook в Интернете (ранее называлась Outlook Web App), admin вашей организации может отключить доступ к Outlook в Интернете. Ваш администратор можете устранить эту ошибку, выполнив действия, описанные в разделе [Включение и отключение Outlook Web App для почтового ящика](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
