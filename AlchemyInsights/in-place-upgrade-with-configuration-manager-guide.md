# <a name="in-place-upgrade-with-configuration-manager-guide"></a>Обновление на месте с руководством Configuration Manager

Процесс обновления на месте сохраняет все данные, параметры, приложения и драйверы из существующей версии операционной системы. Он является чрезвычайно надежным и имеет возможность автоматически вернуть предыдущую операционную систему в случае возникновения проблем.

Используйте [обновление на месте с руководством Configuration Manager](https://admin.microsoft.com/adminportal/home#/win10upgrade) при обновлении устройств с ОС Windows 7 и Windows 8,1 до последней версии Windows 10. Вы будете использовать сценарий, предоставленный для проверки необходимых компонентов и автоматической настройки обновления на месте.
