# <a name="files-restore-feature"></a>Функция восстановления файлов

Если много ваших файлов OneDrive или SharePoint удалено, перезаписано, повреждено или заражено вредоносными программами, можно восстановить сразу все хранилище OneDrive или библиотеку SharePoint до предыдущей версии с помощью функции восстановления файлов. См. также:

- [Восстановление библиотеки OneDrive](https://support.office.com/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a150)

- [Восстановление библиотеки документов](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
