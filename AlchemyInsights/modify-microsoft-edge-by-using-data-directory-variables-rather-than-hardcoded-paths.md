# <a name="modify-microsoft-edge-by-using-data-directory-variables-rather-than-hardcoded-paths"></a>Изменение Microsoft Edge с использованием переменных каталога данных вместо жестко заданных путей

Например, в Windows для хранения данных профиля в локальных данных приложения пользователя, а не в расположении по умолчанию, установите для политики **усердатадир** значение **$ {local_app_data} \едже\профиле**. 

Чтобы узнать больше, ознакомьтесь со статьей [Создание переменных каталога пользовательских данных Microsoft Edge](https://docs.microsoft.com/deployedge/edge-learnmore-create-user-directory-vars).
