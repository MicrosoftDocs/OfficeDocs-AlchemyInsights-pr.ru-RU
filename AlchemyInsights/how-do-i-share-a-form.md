# <a name="share-a-form"></a>Совместное использование формы

Чтобы предоставить общий доступ к викторине или форме в Microsoft Forms, ознакомьтесь со статьей [общий доступ к форме для совместной работе](https://support.office.com/article/Share-a-form-to-collaborate-d5bb5cf0-8401-4c15-bb8c-8e108cd7e69b).

Чтобы управлять возможностью доступа к внешним формам, ознакомьтесь со статьей [Настройка Microsoft Forms](https://support.office.com/article/set-up-microsoft-forms-cc52287a-4550-464d-9a1b-457bf9df2240). 

Чтобы получить сводную ссылку для предоставления общего доступа к результатам, выполните следующие действия.

1. На вкладке **ответы** нажмите кнопку с многоточием (**...**).
3. Выберите команду **создать сводную ссылку**.
4. Нажмите кнопку **Копировать**.

Пользователи, у которых есть доступ к этой ссылке, могут просматривать сводку по ответам на форму.
