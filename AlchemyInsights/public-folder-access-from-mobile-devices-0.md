# <a name="public-folder-access-from-mobile-devices"></a>Доступ к общим папкам с помощью мобильных устройств

Общие папки не доступны с мобильных устройств.
  
Необходимо иметь функциональные возможности, аналогичные общих папок и получить к нему доступ с мобильных устройств, в статье [сведения о группах Office 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) для лучший вариант.
