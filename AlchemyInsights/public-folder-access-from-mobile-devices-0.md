# <a name="public-folder-access-from-mobile-devices"></a>Доступ к общедоступным папкам с мобильных устройств

Общедоступные папки недоступны с мобильных устройств.
  
Если вы хотите использовать функции, схожие с общими папками, и получать к ней доступ с мобильных устройств, ознакомьтесь со статьей " [сведения о группах Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) " для улучшения альтернативы.
