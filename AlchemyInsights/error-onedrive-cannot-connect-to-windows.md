# <a name="error-onedrive-cannot-connect-to-windows"></a>Ошибка: OneDrive не удается подключиться к Windows

Чтобы устранить сообщение об ошибке, выполните поиск файла cldflt.sys ли она запущена, и если это не так, включить. 
  
Для получения дополнительных сведений о включении файла cldflt.sys увидеть [OneDrive не удается подключиться к Windows» Ошибка при доступе к файлов в OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
