# <a name="error-onedrive-cannot-connect-to-windows"></a>Ошибка: OneDrive не удается подключиться к Windows

Чтобы устранить сообщение об ошибке, выполните поиск cldflt.sys файла, чтобы убедиться, что он запущен, а если нет, включите его. 
  
Дополнительные сведения о том, как включить файл cldflt.sys, можно найти [в статье onedrive не удается подключиться к Windows "при доступе к файлам в OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
