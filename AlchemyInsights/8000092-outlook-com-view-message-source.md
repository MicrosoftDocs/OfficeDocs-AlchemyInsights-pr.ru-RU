# <a name="how-to-view-message-headers-or-message-source"></a>Просмотр заголовков сообщений или источника сообщений

Чтобы просмотреть заголовки и другие сведения в сообщении электронной почты, выберите три точки в верхнем углу области чтения и выберите **Просмотр источника сообщения**.
