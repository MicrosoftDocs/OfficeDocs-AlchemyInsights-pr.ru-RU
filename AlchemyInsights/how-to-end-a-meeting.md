# <a name="how-to-formally-end-a-meeting"></a>Как формально завершать собрание

Организаторов собраний может завершить собрание для всех участников в Teams, нажав кнопку **завершить собрание** в элементах управления собрания в собрании.  

Чтобы завершить выполняющееся собрание для всех участников, выберите элемент элементы управления собрания, щелкните значок 3 точки и щелкните закончить собрание. Вам будет предложено подтвердить выбор. Подтверждение выбора приведет к завершению собрания для всех пользователей.

NOTE: только организатор собрания может "завершить" собрание.
