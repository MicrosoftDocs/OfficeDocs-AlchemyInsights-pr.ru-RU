# <a name="powerpoint-slides-not-showing-in-skype-for-business-on-android-devices"></a>Слайды PowerPoint не отображаются в Skype для бизнеса на устройствах с Android

Неполадки с отображением слайдов PowerPoint на устройствах с Android могут быть связаны с ошибкой в браузере Chrome.

Попробуйте отключить, а затем удалить Chrome. После удаления Chrome снова включите Chrome, чтобы получить заводскую версию (74.0.3729.136), в которой эта проблема устранена.
