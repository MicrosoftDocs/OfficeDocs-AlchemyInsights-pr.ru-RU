# <a name="stalleddueto-errors-during-migration"></a>StalledDueTo ошибки во время миграции

Вы можете увидеть различные ошибки, которые начинаются с «StalledDueTo...» во время миграции в Exchange Online. Многие из этих ошибок, совершенно нормальной. Для получения дополнительных сведений см [на основе регулирование ресурсов и определение приоритетов в Exchange Online миграции](https://blogs.technet.microsoft.com/exchange/2018/06/25/resource-based-throttling-and-prioritization-in-exchange-online-migrations/).
