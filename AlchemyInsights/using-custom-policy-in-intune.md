# <a name="using-custom-policy-in-intune"></a>Использование настраиваемой политики в Intune

У Intune есть множество параметров, встроенных для устройств с Android, iOS, macOS и Windows. Вы также можете создать настраиваемый профиль, а затем развернуть эти настраиваемые параметры на устройствах с помощью Intune. Настраиваемые профили предназначены для добавления параметров и компонентов устройств, не встроенных в Intune.

[Настраиваемые параметры для устройств с Android в Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Настраиваемые параметры для устройств Android Enterprise в Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Настраиваемые параметры для устройств с iOS в Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Настраиваемые параметры для устройств macOS в Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Пользовательские параметры для устройств с Windows 10 в Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Настраиваемые параметры для Windows holographic для бизнес-устройств в Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
