# <a name="teams-client-is-stuck-on-loading"></a>Клиент Teams завис при загрузке

Корпорация Майкрософт знает об ошибке в последней сборке классических приложений Teams, которая не позволяет пользователям войти в систему. Отображенное сообщение "Загрузка Microsoft Teams" отображается для пользователей, для пользователей с влиянием на них.

Самый простой обходной путь на этот момент — использовать веб-приложение Teams по пути <https://teams.microsoft.com/> .

Дополнительные сведения и текущее состояние см. в публикации панели мониторинга "Состояние службы" по <https://admin.microsoft.com/AdminPortal/Home#/servicehealth/advisories/:/alerts/TM236316>
