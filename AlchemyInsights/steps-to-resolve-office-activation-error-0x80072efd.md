# <a name="steps-to-resolve-office-activation-error-0x80072efd"></a>Действия по устранению ошибки активации Office 0x80072EFD


Повторите эти решения для исправления ошибок активации Office:
  
1. Проверка [брандмауэра](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckFirewall).
    
2. Проверка [антивирусной программы](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckAV).
    
3. Проверьте [Параметры прокси-сервера](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckProxy)
    
4. Загрузите и установите [поддержки и помощник по восстановлению](https://aka.ms/SARA-OfficeActivation-Alchemy) и запустите **Средство устранения неполадок с активацией**.
