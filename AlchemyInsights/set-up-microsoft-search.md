# <a name="set-up-microsoft-search"></a>Настройка Поиска (Майкрософт)

Microsoft Search‎ provides a user-friendly interface to search and find results in your organization across ‎Microsoft 365‎ services and applications to help you complete what you're working on. It is included at no additional cost with your eligible ‎Microsoft 365‎ subscription.

Используйте руководство по настройке microsoft [Search,](https://go.microsoft.com/fwlink/?linkid=2156919) чтобы настроить, следует ли пилотировать его группе пользователей или выкатывать ее всем в организации. Вы назначите администраторам поиска и редакторам поиска, а затем настроите интерфейс поиска для пользователей с помощью ответов и дополнительных параметров, таких как добавление расширения Bing в Chrome или настройка Bing в качестве поисковой системы по умолчанию.
