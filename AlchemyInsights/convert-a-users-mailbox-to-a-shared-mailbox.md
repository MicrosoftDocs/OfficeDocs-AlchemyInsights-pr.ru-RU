# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Преобразование почтового ящика пользователя в общий почтовый ящик

При преобразовании почтового ящика Office 365 отдельного пользователя в общий почтовый ящик все существующие элементы электронной почты и календаря сохраняются в общем почтовом ящике, к которому у нескольких людей есть доступ. Вы также можете преобразовать общий почтовый ящик обратно в почтовый ящик пользователя (частный).
  
Дополнительные сведения приведены в статье [Преобразование почтового ящика пользователя в общий почтовый ящик](https://docs.microsoft.com/office365/admin/email/convert-user-mailbox-to-shared-mailbox).
