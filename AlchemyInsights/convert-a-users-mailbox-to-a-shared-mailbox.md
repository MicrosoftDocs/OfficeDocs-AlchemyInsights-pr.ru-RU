# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Преобразование почтового ящика пользователя в общий почтовый ящик

При преобразовании почтового ящика пользователя к общему почтовому ящику, сохраняются все существующие электронная почта и календарь. Только теперь находится в общем почтовом ящике, где несколько сотрудники будут иметь доступ вместо одного пользователя. В нужный момент можно преобразовать общего почтового ящика к почтовому ящику пользователя (закрытый).
