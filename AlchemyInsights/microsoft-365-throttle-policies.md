# <a name="microsoft-365-throttle-policies"></a>Политики регулирования Microsoft 365

В Microsoft 365 используется регулирование для обеспечения оптимальной производительности и надежности. Регулирование ограничивает количество действий пользователя или одновременных звонков (по сценарию или коду), чтобы предотвратить чрезмерное использование ресурсов. Дополнительные сведения о регулировании:

- [Ограничения ресурсов Office 365](https://docs.microsoft.com/office365/Enterprise/office-365-resource-limits)

- [Рекомендации по миграции электронной почты Office 365 в отношении регулирования](https://docs.microsoft.com/exchange/mailbox-migration/office-365-migration-best-practices#office-365-throttling)

- [Как избежать регулирования или блокировки в SharePoint Online](https://docs.microsoft.com/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online)
