# <a name="export-user-account-information"></a>Экспорт сведений о учетных записях пользователей

Чтобы экспортировать сведения об учетных записях пользователей в центре администрирования Microsoft 365, включая сведения о лицензиях, политике паролей, заблокированных учетных записях, состоянии синхронизации AAD и т. д., перейдите на страницу Пользователи Активные пользователи, а затем выберите экспорт пользователей   >  [](https://go.microsoft.com/fwlink/p/?linkid=834822) .
