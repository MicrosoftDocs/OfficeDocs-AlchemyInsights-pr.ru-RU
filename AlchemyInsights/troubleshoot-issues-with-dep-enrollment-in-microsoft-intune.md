# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>Устранение неполадок с DEP регистрации в Microsoft Intune

Просмотрите ресурсы, перечисленные ниже, чтобы устранить проблему сейчас. 
  
1. Если устройство DEP не удалось зарегистрировать и поддержкой многофакторной проверкой Подлинности (многофакторной проверки подлинности), отключите многофакторной проверкой Подлинности. В настоящее время многофакторной проверкой Подлинности не поддерживается для регистрации DEP
    
2. [Устранение неполадок в портал Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) используйте для диагностики и устранения распространенных ошибок регистрации. Просмотрите [Этот документ](https://docs.microsoft.com/en-us/intune/help-desk-operators) для получения дополнительных сведений. 
    
3. Просмотрите эти документы, список распространенные ошибки, которые препятствуют регистрации и решения в каждое: [руководство по Устранение неполадок](https://support.microsoft.com/en-us/help/4039809/troubleshooting-ios-device-enrollment-in-intune) и [Устранение неполадок документов](https://docs.microsoft.com/en-us/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)
    
4. [Дополнительные сведения о программе регистрации устройствами](https://docs.microsoft.com/en-us/intune/device-enrollment-program-enroll-ios).
