# <a name="how-to-select-all-in-outlookcom"></a>Как "выделить все" в Outlook.com

Все сообщения можно выбирать только в том случае, если все сообщения являются видимыми. Это необходимо для предотвращения случайных действий в сообщениях, которые не отображаются.

**Удерживая нажатой клавишу CTRL** , вы выделяете несколько отдельных сообщений.

**SHIFT + щелчок** выбирает все сообщения между первым и последним выбранным сообщением.
