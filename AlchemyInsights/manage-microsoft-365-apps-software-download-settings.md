# <a name="manage-office-installation-options"></a>Управление настройками установки Office

Администратор Office 365 может контролировать программное обеспечение Office, которое пользователям разрешено скачивать из Office 365 и устанавливать.  Дополнительные сведения см. в статье [Управление параметрами установки Office в Центре администрирования Microsoft 365](https://docs.microsoft.com/deployoffice/manage-software-download-settings-office-365).
