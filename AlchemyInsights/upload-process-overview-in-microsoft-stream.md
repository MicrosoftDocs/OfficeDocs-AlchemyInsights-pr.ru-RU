# <a name="upload-process-overview-in-microsoft-stream"></a>Обзор процесса отправки в Microsoft Stream

Когда вы отправляете видео, Microsoft Stream автоматически кодирует его на несколько скорость, чтобы его можно было потокировать с помощью адаптивной потоковой передачи по каналу. Microsoft Stream оптимизирует скорость и качество воспроизводимого видео в зависимости от доступной пропускной способности средства просмотра.

- [Обзор процесса отправки в Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
