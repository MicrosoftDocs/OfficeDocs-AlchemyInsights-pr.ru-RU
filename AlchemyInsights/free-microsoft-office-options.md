# <a name="free-microsoft-office-options"></a>Бесплатные варианты Microsoft Office

Вы можете бесплатно создавать, делиться и совместно работать с помощью веб-версий Microsoft Word, PowerPoint, Excel и OneNote.

Другие бесплатные варианты, которые можно использовать:

- [Приложения Microsoft Office для мобильных устройств](https://products.office.com/mobile/office?wt.mc_id=Cons_Office_Chatbot). Используйте приложения Office для мобильных устройств (iPhone, iPad, телефоны и планшеты с Android) с размером экрана менее 10,1 дюйма.

- Учащиеся соответствующих учебных заведений могут [зарегистрироваться для получения Microsoft 365 для образования бесплатно](https://www.microsoft.com/education/products/office?wt.mc_id=Cons_Office_Chatbot).

- Некоторые новые компьютеры получают бесплатный год использования Office. Дополнительные сведения см. в статье [Активация платной или бесплатной версии Office на новом компьютере](https://support.office.com/article/89881633-0b26-4ca8-816b-93f347bd92c0?wt.mc_id=Cons_Office_Chatbot).
