# <a name="changing-public-folder-permissions"></a>Изменение разрешений для общедоступных папок

Вы и ваши пользователи могут изменять разрешения общеДоступных папок в Outlook. Просмотр [или изменение разрешений для общедоступной папки](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). Кроме того, администраторы могут изменять разрешения в центре администрирования Exchange:
  
- в центре администрирования Microsoft 365 перейдите в центр **администрирования** \> **Exchange**.
    
- Выберите пункт общеДоступные **папки**.
    
В этом случае вы можете изменить разрешения для отдельных общедоступных папок, назначив пользователям группы безопасности разрешения. Чтобы изменить разрешения для общедоступной папки, необходимо быть ее владельцем.
