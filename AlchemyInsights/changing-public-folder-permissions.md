# <a name="changing-public-folder-permissions"></a>Изменение разрешений для общей папки

Пользователи могут изменить разрешениями для общих папок в Outlook. Просмотреть [установить или изменить разрешения для общей папки](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). «Администраторы» можно также изменить разрешения в центре администрирования Exchange:
  
- В центре администрирования Office 365 перейдите на **центры администрирования** \> **Exchange**.
    
- Выберите **Общие папки**.
    
Отсюда можно изменить разрешения для отдельных общих папок, назначение пользователей группам безопасности для разрешений. Чтобы изменить разрешения для общей папки, необходимо быть владельцем.
