# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Как работает наследование разрешений в SharePoint

По умолчанию разрешения в SharePoint наследуются выше в иерархии. Таким образом, файл наследует разрешения из папки, которая наследует разрешения из библиотеки, которая наследует свои разрешения от сайта, который наследует разрешения от семейства веб-сайтов.
  
Сведения об удалении уникальных разрешений и восстановлении наследования можно найти [в разделе Изменение разрешений для списка или библиотеки и управление ими](https://go.microsoft.com/fwlink/?linkid=869946).
