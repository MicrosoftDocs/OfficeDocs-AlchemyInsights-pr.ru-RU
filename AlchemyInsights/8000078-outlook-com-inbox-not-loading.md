# <a name="inbox-not-loading-with-firefox"></a>Папка "Входящие" не загружается с помощью Firefox

Некоторые надстройки в браузере Firefox могут препятствовать загрузке папки "Входящие" Outlook.com.
  
Посетите страницу поддержка Firefox, чтобы узнать, как [отключить или удалить надстройки](https://support.mozilla.org/kb/disable-or-remove-add-ons).
