# <a name="teams-meeting-recordings-to-onedrive"></a>Записи собраний Teams в OneDrive

В Microsoft Teams существует новый метод сохранения записей собраний. На первом этапе перехода с классической версии Microsoft Stream на [новый Stream](https://docs.microsoft.com/stream/streamnew/new-stream) этот метод сохраняет записи в Microsoft OneDrive для бизнеса и SharePoint в Microsoft 365, а также предлагает множество преимуществ.  

- [Настройка параметра записи собрания в OneDrive для бизнеса и SharePoint](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#set-up-the-meeting-recording-option-for-onedrive-for-business-and-sharepoint)
- [Отказ от продолжения использования Stream в OneDrive для бизнеса и SharePoint](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#opt-out-of-onedrive-for-business-and-sharepoint-to-continue-using-stream)  
- [Вопросы и ответы](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#frequently-asked-questions)
