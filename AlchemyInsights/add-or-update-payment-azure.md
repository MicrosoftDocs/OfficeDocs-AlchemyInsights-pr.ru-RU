# <a name="update-billing-info-under-azure-enterprise-agreement"></a>Обновление сведений об оплате в соглашении Azure Enterprise

Изменение сведений об оплате для подписки Azure Enterprise не поддерживается.

Если вы хотите изменить адрес в существующей учетной сумме, свяжитесь с представителем корпорации Майкрософт.
