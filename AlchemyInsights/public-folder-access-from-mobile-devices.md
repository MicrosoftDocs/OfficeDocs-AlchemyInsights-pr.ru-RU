# <a name="public-folder-access-from-mobile-devices"></a>Доступ к общедоступным папкам с мобильных устройств

Доступ к общедоступным папкам не работает на мобильных устройствах. для доступа к общедоступным папкам Используйте OWA (Outlook в Интернете) или Outlook для настольных компьютеров Windows или Outlook для Mac.

Если вы хотите иметь функциональные возможности, аналогичные общедоступным папкам и имеющие доступ к контенту на мобильных устройствах, ознакомьтесь со статьей [Microsoft 365 Groups](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) , чтобы лучше.
