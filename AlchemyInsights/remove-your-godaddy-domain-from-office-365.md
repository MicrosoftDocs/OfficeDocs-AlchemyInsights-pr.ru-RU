# <a name="remove-your-godaddy-domain-from-microsoft-365"></a>Удаление домена GoDaddy из Microsoft 365

Мы не можем удалить ваш домен GoDaddy из Microsoft 365. Обратитесь за помощью в [службу поддержки GoDaddy](https://aka.ms/contact-godaddy) .
  
Если название организации не содержит "НЕТОРГ" или не было приобретено в GoDaddy, удалите домен, выполнив следующие действия [. Удалите домен из Microsoft 365.](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain)
