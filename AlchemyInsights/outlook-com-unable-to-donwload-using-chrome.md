# <a name="cant-download-files-in-chrome"></a>Не удается скачать файлы в Chrome

**Ошибка** У нас возникла ошибка, из-за которой некоторые клиенты не смогут загружать файлы при использовании браузера Google Chrome. 

**Обходной путь** Используйте [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) или другой веб-браузер.
Узнайте больше о том, как [загружать файлы и вложения Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
