# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Устранение неполадок установки мобильного Outlook для общего почтового ящика

Не может получить доступ к общему почтовому ящику из приложения Outlook, которая устанавливается на мобильном устройстве. Вы хотите, чтобы этот компонент? Пусть знать команды разработчиков Outlook. [Здесь голос](https://go.microsoft.com/fwlink/?linked=862116)!
  
Можно использовать только общего почтового ящика на мобильном устройстве, откройте браузер, вход в Office 365 и перейдите Outlook в Интернете. Для получения инструкций в разделе [Add общего почтового ящика в Outlook в Интернете](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Из Outlook в Интернете вы сможете получить доступ к общему почтовому ящику.
