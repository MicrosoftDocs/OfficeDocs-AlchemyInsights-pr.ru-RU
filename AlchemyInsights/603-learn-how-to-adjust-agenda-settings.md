# <a name="learn-how-to-configure-agenda-settings"></a>Сведения о настройке параметров повестки дня

Параметры повестки дня, находятся в параметры **календаря** в Outlook в Интернете. Для получения дополнительных сведений см. [Параметры календаря в Outlook в Интернете](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).
  
Администраторы могут использовать командлет **Set-CalendarNotification** в Exchange Online PowerShell, чтобы настроить параметры повестки дня. Для получения дополнительных сведений см [Set-CalendarNotification](https://technet.microsoft.com/library/dd351284).
