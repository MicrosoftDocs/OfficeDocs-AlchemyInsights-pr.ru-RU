# <a name="submit-dynamics-365-support-requests"></a>Подавать запросы на поддержку Dynamics 365

Если вы являетесь клиентом **Сотрудничества с клиентом Microsoft Dynamics** отправьте вашей билетов через новые [платформы Power центра администрирования](https://admin.powerplatform.microsoft.com/?ref=officemodern).
  
Новый портал предоставляет более прямой путь к интерактивной поддержки, который оптимизирован для сотрудничества с клиентами Dynamics и других продуктов питания платформы.
