# <a name="long-running-script"></a>Длительный Скрипт

Мы осведомлены о проблемах с "долго выполняемым сценарием" в Outlook.com. Чтобы устранить эту проблемы, используйте бета-версию Outlook.com.
  
[Узнайте больше о бета-версии Outlook.com.](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)
