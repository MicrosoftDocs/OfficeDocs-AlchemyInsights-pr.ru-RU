# <a name="data-location"></a>Расположение данных

Вы можете просмотреть расположение клиента в центре администрирования, перейдя на [страницу **профиля организации** ](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) и выбрав **расположение данных**.

Для приложений, на которые вы не подписаны, ознакомьтесь со статьей, в [которой хранятся данные клиента Microsoft 365](https://docs.microsoft.com/office365/enterprise/o365-data-locations).
