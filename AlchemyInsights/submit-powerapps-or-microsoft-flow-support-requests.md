# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Передача запросов PowerApps или поддержки Microsoft Flow

Если вы являетесь клиентом **PowerApps** или **Microsoft Flow** , отправьте свой билет через новый [центр администрирования платформы управления питанием](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819). Новый портал предоставляет более прямой путь к технической поддержке, оптимизированной для PowerApps и других продуктов Power Platform.

[Отправить запрос в службу поддержки Microsoft Flow](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
